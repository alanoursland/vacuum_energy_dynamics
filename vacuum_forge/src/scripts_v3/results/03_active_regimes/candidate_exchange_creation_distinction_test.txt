Case 0: P1 + P3 bookkeeping identity
========================================================================================================
P1 + P3 bookkeeping:
  E_vac = rho_v V_vac
  rho_v = constant

delta E_vac = dV*rho_v

Therefore:
  delta E_vac = 0  <=>  delta V_vac = 0
when rho_v is nonzero and constant.

[PASS] energy preservation and volume preservation are equivalent under P1+P3
But this does not decide whether a process is exchange or creation.
That classification must be supplied or derived separately.

Case 3: Relaxation operator as energy descent
========================================================================================================
E = C_k*kappa**2 + C_s*s**2
grad_kappa = 2*C_k*kappa
grad_s = 2*C_s*s
d kappa/dtau = -2*C_k*kappa
d s/dtau = -2*C_s*s
dE/dtau = -4*C_k**2*kappa**2 - 4*C_s**2*s**2
[PASS] relaxation decreases energy away from equilibrium

Equilibrium of unsourced relaxation:
  grad_kappa = 0 -> kappa=0
  grad_s = 0 -> s=0

For exterior gravity, s remains nonzero only if boundary/source data
or a shear source law maintains it. Kappa can still relax to zero.
[PASS] unsourced relaxation drives kappa toward zero

Case 6: Impact on P3 volume-preservation question
========================================================================================================
The earlier P3 volume-preservation question was:

  Does P3 force exchange to preserve volume?

The reformulated answer is:

  P1+P3 make energy preservation equivalent to volume preservation.
  But P1+P3 do not decide whether a process is exchange or creation.

This script therefore treats exchange/creation/destruction as operator
classes and checks their reduced consequences.

Result:
  If exchange is defined as conservative redistribution, then it is
  trace-kernel and gives J_kappa=0 in the reduced model.

  Creation/destruction are traceful and may source kappa.

Open question:
  Can the exchange/creation distinction be derived from the postulates,
  or must it be added as a structural principle?
[PASS] P3 alone does not classify process regime

Final interpretation
========================================================================================================
This reduced operator test supports the regime classification:

  exchange:
    conservative redistribution
    delta E = delta V = 0 under P1+P3
    J_kappa = 0
    kappa_eq = 0
    AB = 1

  creation/destruction:
    net vacuum amount change
    J_kappa != 0 generically
    kappa_eq != 0 generically
    AB != 1 generically

  relaxation:
    energy descent / fill-in response
    can drive kappa toward zero
    can coexist with boundary-maintained shear

This does not prove which regimes nature uses.
It only confirms that the proposed classification is algebraically
consistent with the reduced kappa/s exterior program.
