Case 0: Global rotation mode problem
========================================================================================================================
The transverse projector uses 1/k^2 and works for local k != 0 modes.

But global rotation / total angular momentum is a boundary/global issue.

  How should a compact rotating source set exterior vector boundary data?

Rules:

  keep coefficients symbolic

  treat global angular momentum as boundary/source data

========================================================================================================================
Case 1: Angular momentum from current
========================================================================================================================
Current:
j = Matrix([[rho*v_x], [rho*v_y], [rho*v_z]])

Angular momentum density:
ell = r x j =
Matrix([[rho*(-v_y*z + v_z*y)], [rho*(v_x*z - v_z*x)], [rho*(-v_x*y + v_y*x)]])

Global angular momentum:

  J = integral r x j d^3x

Interpretation:
  A rotating source supplies global vector boundary data through J.

========================================================================================================================
Case 2: Uniform rotation current
========================================================================================================================
Rigid rotation about z:
v = Omega x r =
Matrix([[-Omega*y], [Omega*x], [0]])

j = rho v =
Matrix([[-Omega*rho*y], [Omega*rho*x], [0]])

div j = 0

For constant rho and Omega, the current is divergence-free.

Case 3: Boundary source view
========================================================================================================================
For a compact rotating source of radius R:

  interior current determines total angular momentum J
  exterior vector solution should be fixed by boundary data at R

Symbolic boundary condition:

  vector boundary circulation/flux ~ C_b J

boundary data = C_b*J

C_b is not derived.
