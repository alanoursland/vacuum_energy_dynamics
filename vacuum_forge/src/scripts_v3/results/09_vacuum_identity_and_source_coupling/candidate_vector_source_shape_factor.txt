Case 3: Total current monopole vanishes
========================================================================================================================
For a symmetric uniformly rotating sphere:

  integral j_x d^3x = -rho Omega integral y d^3x = 0
  integral j_y d^3x =  rho Omega integral x d^3x = 0
  integral j_z d^3x = 0

Therefore the current monopole vanishes.
The leading far-field vector effect is dipole/angular-momentum-like.

========================================================================================================================

J_z = I_z Omega = 8*pi*Omega*R**5*rho/15

So:

  J_z = (2/5) M R^2 Omega
