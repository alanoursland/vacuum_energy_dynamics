Final interpretation
============================================================================================
This source-law toy establishes the reduced exterior chain:

  kappa = 0
    -> A = exp(s), B = exp(-s), AB = 1

  source-free exterior shear equation:
    ∇²s = 0

  spherical solution plus asymptotic flatness:
    s(r) = C/r

  mass/interface flux condition:
    4πr²s'(r) = 8πGM/c²

  coefficient:
    C = -2GM/c²

  result:
    s(r) = -2GM/(r c²)

  metric:
    A = exp(s) ≈ 1 - 2GM/(r c²)
    B = exp(-s) ≈ 1 + 2GM/(r c²)
    AB = 1 exactly

This does NOT yet derive the field equation or the mass/interface flux law.
It shows that if the shear mode obeys a Laplace/Poisson source law with
the stated normalization, then the weak-field exterior profile follows.
