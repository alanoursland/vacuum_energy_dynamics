Case 0: Kappa pressure/trace model problem
========================================================================================================================
Question:

  If kappa is sourced by pressure or spatial stress trace, does an
  interior matter body generate forbidden exterior kappa charge?

Toy source:

  S_trace = 3p

Goal:

  determine whether pressure trace requires projection/screening to avoid
  an exterior 1/r kappa tail.
[WARN] pressure trace model problem posed: CONSTRAINED_BY_IDENTITY

========================================================================================================================
Case 1: Simple pressure profile
========================================================================================================================
Toy pressure profile:

  p(r) = p0 (1 - r^2/R^2)

p(r) = p0*(1 - r**2/R**2)

Spatial trace source:

  S_trace = 3p

S_trace = 3*p0*(1 - r**2/R**2)

This is a toy interior source, not a GR pressure profile.

========================================================================================================================
Case 2: Integrated trace charge
========================================================================================================================
Integrated trace charge:

  Q_kappa = 4*pi integral_0^R r^2 S_trace dr

Q_kappa = 8*pi*R**3*p0/5

For p0 > 0, Q_kappa is nonzero.

If massless kappa uses this as ordinary charge, exterior kappa gets a 1/r tail.
[WARN] pressure trace has nonzero integrated charge: RISK — massless kappa would leak outside

========================================================================================================================
Case 3: Massless exterior flux consequence
========================================================================================================================
Schematic massless equation:

  -K_k Delta kappa = alpha_k S_trace

Exterior flux proportional to integrated trace charge:

F_kappa ~ -Q_kappa*alpha_k/K_k

Exterior tail:

kappa_ext ~ Q_kappa*alpha_k/(4*pi*K_k*r)

Thus a positive pressure trace source generically creates exterior leakage.
[WARN] massless pressure-sourced kappa leaks outside: RISK — requires projection, screening, or zero-charge construction

========================================================================================================================
Case 4: Zero-charge requirement
========================================================================================================================
To preserve exterior kappa = 0 with massless kappa:

  Q_kappa = integral S_trace d^3x must vanish

But for ordinary positive pressure:

  integral 3p d^3x > 0

Therefore a raw pressure-trace Poisson law is not exterior-safe.

Needed:

  constraint projection that removes monopole kappa charge
  or restoring/massive suppression
  or source defined as compensated trace exchange with zero net charge
[WARN] zero-charge condition conflicts with raw positive pressure: RISK — raw S_trace=3p not enough

========================================================================================================================
Case 5: Compensated trace source option
========================================================================================================================
Compensated source idea:

  S_comp = S_trace - <S_trace>

<S_trace> = 6*p0/5

S_comp = 9*p0/5 - 3*p0*r**2/R**2

Integrated compensated charge:

Q_comp = 0

This removes monopole kappa charge by construction.
But it must be derived from a parent constraint, not inserted arbitrarily.
[WARN] compensated trace source can enforce zero charge: CONSTRAINED_BY_IDENTITY — needs parent identity

========================================================================================================================
Case 6: Massive/restoring trace response option
========================================================================================================================
If kappa has a restoring/mass scale:

  (-Delta + m_k^2) kappa = source

then exterior tail is Yukawa-suppressed:

kappa_ext ~ Q*alpha_k*exp(-m_k*r)/(4*pi*K_k*r)

This allows nonzero integrated trace charge but avoids long-range leakage.

However m_k must be derived or constrained.
[WARN] massive trace response can suppress leakage: PLAUSIBLE — new scale m_k missing

========================================================================================================================
Final interpretation
========================================================================================================================
A raw pressure/spatial-trace source is plausible for the interior, but
dangerous for the exterior.

For positive pressure:

  Q_kappa = integral 3p d^3x != 0

A massless kappa equation would then produce:

  kappa_ext ~ 1/r

Therefore raw pressure trace cannot be the final unscreened kappa source.

Need one of:

  compensated/zero-charge trace constraint
  massive/restoring suppression
  parent projection that removes exterior kappa charge
