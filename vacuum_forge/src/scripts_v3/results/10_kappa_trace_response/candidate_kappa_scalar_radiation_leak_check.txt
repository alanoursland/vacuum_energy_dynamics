Case 2: Massless hyperbolic wave hazard
========================================================================================================================
For a massless scalar wave equation:

  Box kappa = 0

plane-wave modes obey:

Eq(omega**2, c**2*k**2)

Even if the monopole source is projected out, time-dependent trace
structure can excite propagating scalar modes.

This is not acceptable as an ordinary long-range gravity wave channel.

========================================================================================================================
Case 3: Elliptic constraint safety
========================================================================================================================
If kappa is elliptic/constrained:

  L_kappa kappa = alpha_k P_0 S_trace

with no independent second time derivative:

  no Box kappa term

then kappa is determined instantaneously/constraint-wise by source and
boundary/projection data.

This avoids an independent scalar radiation channel.

But it requires a parent constraint identity to be legitimate.

========================================================================================================================
Case 4: Massive/restoring dispersion
========================================================================================================================
A massive/restoring scalar mode would have schematic dispersion:

Eq(omega**2, c**2*k**2 + m**2)

This suppresses long-range low-frequency propagation if m is large enough
or if kappa is not freely radiative.

But it still introduces a scalar mode unless the dynamics are constrained
or strongly damped.

========================================================================================================================
Case 5: Relaxation safety
========================================================================================================================
Relaxational trace deviation:

  dot kappa = -Gamma kappa

solution:

kappa(t) = kappa_0*exp(-Gamma*t)

This absorbs scalar trace perturbations rather than propagating them.

But relaxation requires energy accounting and a source/restoring basis.
