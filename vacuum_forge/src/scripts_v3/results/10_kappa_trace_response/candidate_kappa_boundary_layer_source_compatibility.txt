Case 4: Compensated trace comparison
========================================================================================================================

C2 effective source:

S_eff = -6*kappa_0*(-R + r)*(R + r)*(-3*R**2 + 7*r**2)/R**6

Both have zero net integral and sign-changing structure.
They are not identical shapes, but they are in the same compensated-source family.

Case 5: Shifted local minimum interpretation
========================================================================================================================
If non-inertial relaxation reaches local equilibrium:

  kappa = kappa_min

and:

  kappa_min = chi_k S_trace_effective

then required effective trace source is:

S_trace_effective = kappa_0*(R**2 - r**2)**3/(R**6*chi_k)

This source is compact, positive for k0/chi_k > 0, and vanishes smoothly at R.

Important:
  this is different from the elliptic effective source Delta kappa.
  in the non-inertial model, trace shifts the minimum rather than acting
  as a Poisson charge.

Case 6: Operator-dependence warning
========================================================================================================================
Source compatibility depends on which equation kappa obeys.

  source ~ Delta kappa

  C2 can come from a compact positive shifted minimum

interior trace response than a Poisson-source kappa model.
