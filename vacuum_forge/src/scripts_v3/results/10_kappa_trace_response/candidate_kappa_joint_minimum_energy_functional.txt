Case 1: Define toy energy density
========================================================================================================================
Toy energy density:

  L = W_int (f-f_int)^2 + W_ext (f-f_ext)^2
      + lambda_1 (f')^2 + lambda_2 (f'')^2

L = lambda_1*Derivative(f(r), r)**2 + lambda_2*Derivative(f(r), (r, 2))**2 + (f(r) - f_ext(r))**2*W_ext(r) + (f(r) - f_int(r))**2*W_int(r)

Interpretation:
  W_int anchors to interior tendency
  W_ext anchors to exterior tendency
  lambda_1 penalizes slope strain
  lambda_2 penalizes curvature jumps / sharp bending

========================================================================================================================
Case 2: Euler-Lagrange equation with second-derivative term
========================================================================================================================
Euler-Lagrange equation for L depending on f, f_prime, f_double_prime:

  dL/df - d/dr(dL/df_prime) + d^2/dr^2(dL/df_double_prime) = 0

Computed structure:

-2*lambda_1*Derivative(f(r), (r, 2)) + 2*lambda_2*Derivative(f(r), (r, 4)) + 2*(f(r) - f_ext(r))*W_ext(r) + 2*(f(r) - f_int(r))*W_int(r)

The lambda_2 term makes the toy minimizer fourth-order.
That is expected for curvature-smoothing energy.

Boundary conditions must therefore be handled carefully.

========================================================================================================================
Case 3: Simplified constant-weight equation
========================================================================================================================
For constant weights, the schematic equation is:

Eq(W_e*(f(r) - f_ext(r)) + W_i*(f(r) - f_int(r)) - lambda_1*Derivative(f(r), (r, 2)) + lambda_2*Derivative(f(r), (r, 4)), 0)

Equivalent:

  lambda_2 fourth_derivative(f) - lambda_1 second_derivative(f)
  + (W_i+W_e) f = W_i f_int + W_e f_ext

This shows the minimizer is pulled toward both tendencies while
smoothness terms distribute the mismatch.

Case 4: Interior/exterior tendencies

Case 5: Weight functions
========================================================================================================================
A smooth model needs radial weights such as:

  W_int(r): large inside, fades near/outside surface
  W_ext(r): small inside, dominates outside

A possible smooth transition variable:

  x = (r - R)/sigma

A possible logistic weight:

  W_ext = 1/(1 + exp(-x))
  W_int = 1 - W_ext

This avoids a hard seam but introduces sigma.

sigma would control the near-boundary deviation width.
