Case 3: Projection idempotence
========================================================================================================================
A true projection should satisfy:

  P_0(P_0 S) = P_0 S

Reason:
  once the support-average is removed, applying the same operation again
  changes nothing.

For compact support with fixed V:

  <P_0 S> = 0
  P_0(P_0 S) = P_0 S - <P_0 S> = P_0 S

Thus P_0 is idempotent if the support region is fixed.
