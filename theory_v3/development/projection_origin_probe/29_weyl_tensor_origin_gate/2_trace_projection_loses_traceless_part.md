# 2. Trace projection loses traceless part

Take the nonzero shear matrix

```text
S = [[s,0],[0,-s]].
```

Its trace is

```text
tr(S) = 0.
```

So scalar trace projection annihilates it even though `S != 0` when `s != 0`.

Closed result:

```text
trace/monopole scalar data cannot reconstruct traceless shear.
```
