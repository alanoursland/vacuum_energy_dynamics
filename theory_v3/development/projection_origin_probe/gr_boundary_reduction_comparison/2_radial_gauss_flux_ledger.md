
# 2. Radial Gauss flux ledger

For the exterior weak-field potential

```text
Φ(r) = -GM/r,
```

the radial flux is

```text
4πr² Φ'(r) = 4πGM.
```

Validated checks:

```text
4πr² d(-GM/r)/dr = 4πGM,
d/dr [4πr² Φ'(r)] = 0.
```

Interpretation: the boundary ledger is the standard enclosing-surface charge
ledger. It is not distinctive to the projection ontology.
