
# 12. Different-R witness

If the GR reduction landed in `R=1` instead of `R=0`, the first ratio would be

```text
R=0: r_(0,1) = 1/5,
R=1: r_(1,1) = 1/7.
```

Validated difference:

```text
r_(1,k) - r_(0,k)
= -2(2k-1)/[(2k+3)(2k+5)].
```

So a genuine contact-class mismatch would be directly visible in the projection
coefficient.
