# Scalar variation boundary potential

The scalar Dirichlet variation has the integration-by-parts split

```text
phi' eta' = -phi'' eta + d(phi' eta)/dx.
```

The boundary potential in this model is `theta = phi' delta phi`. This is the prototype for the later symplectic boundary ledger.
