# Negative Results Index - Proof Chain Plan

## Purpose

This folder collects the negative/no-go and limitation results that have become
load-bearing in the current proof chain. It is not a new origin claim. It is an
index of dead branches, scope limits, and conditional gates.

The goal is to prevent future proof groups from reopening branches that have
already been eliminated or bounded:

```text
scalar ladder != full tensor geometry
scalar boundary charge != radiative symplectic data
scalar nonlinear completion != GR post-Newtonian tensor structure
stress coupling != matter ontology
finite scalar flux != projection contact index R
EH/GHY minimal branch != derivation from r_k alone
```

## Strategy

Each proof script records one negative result or scope boundary and backs it
with an algebraic, component-counting, or symbolic witness.

The folder should close the following ledger:

1. Scalar trace data cannot reconstruct traceless tensor data.
2. Scalar monopole averaging cannot reconstruct higher multipoles.
3. Scalar boundary flux does not encode TT radiation.
4. Scalar nonlinear polynomial dressing does not create tensor degrees of freedom.
5. Conformal/trace-only coupling misses traceless stress.
6. Stress variation does not derive microscopic matter ontology.
7. Finite flux does not determine the projection contact index R.
8. Matching GR's scalar boundary ledger does not prove or disprove ontology work.
9. Nonquadratic directional response cannot be hidden inside a metric branch.
10. EH/GHY minimality is conditional on metric/diffeomorphism/locality/boundary assumptions.
11. Lambda = 0 is an asymptotic branch selector, not a local ontology theorem.
12. Torsion/nonmetricity require explicit routing rather than hidden inclusion.

## Expected conclusion

The project's strongest results include negative results. These do not merely
say what the theory is not; they force the route from scalar admissibility to
metric/tensor/radiative geometry and mark the remaining open gates honestly.
