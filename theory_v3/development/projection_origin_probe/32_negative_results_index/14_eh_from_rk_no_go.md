# 14. EH/GHY From r_k Alone No-Go

The scalar coefficient `r_k` is a boundary/admissibility coefficient. It does
not contain the full assumption set needed for EH/GHY:

```text
metric interval structure,
diffeomorphism redundancy,
second-order locality,
boundary differentiability,
D=4 Lovelock branch,
no extra un-routed fields.
```

## Closed result

```text
r_k alone -/-> EH/GHY.
```

EH/GHY can be selected only after the geometric/action assumptions are present.
