# Executive Summary

The project began with a specific coefficient:

```text
r_k = (2k - 1)/(2k + 3).
```

That coefficient appeared in a scalar projection hierarchy and looked physically suggestive. The early goal was to understand whether it had a deeper origin. The current conclusion is that the original mystery has been solved: `r_k` is the `R = 0` base case of a boundary/admissibility moment-kernel ladder,

```text
r_(R,k) = (2k - 1)/(2k + 2R + 3),
```

and it can be derived through equivalent boundary-reduction routes: moment cancellation, endpoint contact, primitive integration by parts, and compactified scalar boundary pairing.

This is a real success, but it also narrows the meaning of the discovery. The scalar projection ladder is not a standalone physical field equation. It is a scalar boundary/admissibility signature. It captures the trace / monopole / Newtonian boundary sector.

The subsequent proof chain showed why this scalar sector cannot be the whole theory. Scalar boundary data cannot reconstruct traceless shear, Weyl curvature, transverse-traceless radiation, full tensor boundary phase space, matter microstructure, or nonlinear Einstein dynamics. Those require additional structures: directional interval probes, exact quadratic response, metric reconstruction, calibration-coherent transport, matter stress coupling, symplectic boundary data, and constraint closure.

The current GR branch is conditionally closed in the following sense:

```text
At epsilon = 0,
and assuming exact quadratic metric response,
calibration-coherent Levi-Civita transport,
shared metric matter coupling,
diffeomorphism invariance,
second-order locality,
boundary differentiability,
D = 4,
and no extra un-routed fields,

the proof chain reconstructs the standard GR branch:
scalar Newtonian limit,
Weyl/TT tensor sector,
radiative boundary memory,
Hamiltonian/symplectic boundary structure,
nonlinear constraint closure pressure,
and EH+GHY action closure.
```

This is valuable, but it is not yet a derivation of new field equations from the vacuum ontology. The decisive missing object is the vacuum configuration functional, especially its gradient/strain sector.

The best current diagnosis is:

```text
The local part of the ontology supplies pointwise interval response.
The Hessian of that response gives metric-like local data.

The missing gradient/strain part must supply transport, propagation,
curvature dynamics, field equations, radiation, and possible deviations from GR.
```

The next research phase should therefore shift from GR reconstruction to vacuum strain dynamics. The core question is not another form of `r_k`. It is:

```text
What vacuum strain functional determines how neighboring local configurations compare?
```

If the strain functional forces `epsilon = 0`, the project becomes a clean ontology-based rederivation of GR. If it produces `epsilon != 0`, the project becomes a candidate extension of GR with possible physical predictions. If the current ontology cannot determine the strain functional, then the honest conclusion is that a new axiom is required.
