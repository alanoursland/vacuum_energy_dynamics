# Conditional GR Branch Reconstruction

The project now contains a conditional reconstruction of the GR branch.

This means that, after certain gates are imposed, the standard structures of GR appear as the minimal consistent branch. This is not the same as deriving GR from `r_k` alone.

The chain can be summarized as follows.

## 1. Exact quadratic response gives metric data

Directional interval response is represented by a function

```text
Q_p(v).
```

If this response satisfies the exact quadratic/parallelogram gate, polarization reconstructs a symmetric bilinear form:

```text
h(u,v) = (Q(u+v) - Q(u) - Q(v))/2.
```

This is the metric branch.

If nonquadratic response survives, it must be routed as Finsler-like, medium-like, or constitutive extra structure.

## 2. Calibration-coherent transport gives the Levi-Civita branch

Metric compatibility preserves interval comparison under transport.

Torsion and nonmetricity are not mathematically impossible, but they require additional source/defect ledgers. In the absence of such ledgers, the minimal stable branch is Levi-Civita transport.

## 3. Shared metric matter action gives stress coupling

If matter actions use the shared metric interval and volume structure, then metric variation gives the stress tensor source route:

```text
delta S_m ~ (1/2) integral sqrt(-g) T_ab delta g^ab.
```

This does not derive matter ontology. It only closes the coupling route once matter action exists.

## 4. Tensor data gives Weyl/TT sectors

Directional probes recover traceless/shear data that scalar trace ledgers cannot see.

Transport and constraint closure then support Weyl/TT radiative dynamics.

## 5. Boundary symplectic structure gives the dynamical boundary ledger

Scalar Gauss flux is a constraint/charge ledger.

Radiation, memory, and soft shifts live in a tensor/symplectic boundary ledger.

## 6. Nonlinear constraints pressure the Einstein branch

With metric structure, diffeomorphism redundancy, universal stress coupling, second-order locality, boundary differentiability, and no extra fields, the Einstein/Hamiltonian architecture is the minimal known nonlinear closure branch.

## 7. EH+GHY is the minimal action closure under standard gates

In four dimensions, Lovelock minimality leaves the Einstein-Hilbert term as the unique dynamical local curvature term. Boundary differentiability requires a GHY-like completion.

Thus:

```text
metric + diffeomorphism + second-order locality + boundary differentiability
+ D=4 + no extra fields
    -> EH+GHY minimal branch.
```

## What is closed

The GR branch is conditionally reconstructed at

```text
epsilon = 0.
```

At that point, the project can reproduce the expected scalar, tensor, radiative, boundary, constraint, and action structures of GR.

## What is not closed

The project has not yet shown that the vacuum ontology itself forces

```text
epsilon = 0.
```

Nor has it yet specified the vacuum strain functional that generates transport and field equations.

Therefore the current status is:

```text
GR branch: conditionally reconstructed.
Vacuum field equations: not yet derived.
```
