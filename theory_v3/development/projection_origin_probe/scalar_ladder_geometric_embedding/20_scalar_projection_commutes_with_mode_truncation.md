# 20. Scalar Projection Commutes With Mode Truncation

Whether one first truncates to `N` boundary modes and then takes the scalar
trace, or first regards each mode as carrying a scalar trace channel, the scalar
ledger contains `N` coefficients.

The tensor ledger does not collapse this way; it contains `N m(m+1)/2`
coefficients. This separates scalar-mode truncation from tensor completion.
