# 4. Off-Diagonal Scalar Invisibility

An off-diagonal symmetric response

```text
H = [[0, k], [k, 0]]
```

has zero trace. The scalar channel cannot detect it. But polarization from
directional probes gives

```text
H_12 = (Q(e_1 + e_2) - Q(e_1) - Q(e_2))/2 = k.
```

Thus off-diagonal metric data belongs to the directional branch, not the scalar
ladder.
