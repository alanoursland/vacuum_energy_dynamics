# 17. TT Mode Scalar Projection Zero

A standard transverse-traceless block

```text
[[h_plus, h_cross, 0],
 [h_cross, -h_plus, 0],
 [0, 0, 0]]
```

has zero trace, so scalar projection kills it. Directional polarization still
recovers `h_plus` and `h_cross`. Therefore TT radiation cannot be supplied by
the scalar ladder alone.
