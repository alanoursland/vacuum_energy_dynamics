# 14. Per-Mode Scalar Tensor Gap

For `N` boundary modes, the scalar ladder supplies `N` scalar coefficients. A
full symmetric tensor boundary response supplies

```text
N m(m+1)/2
```

coefficients. The missing tensor data is therefore

```text
N (m(m+1)/2 - 1).
```

This proves that increasing scalar mode count alone does not close the tensor
rank gap.
