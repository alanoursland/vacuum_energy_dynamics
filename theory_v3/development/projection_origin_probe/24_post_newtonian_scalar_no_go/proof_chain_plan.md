# Post-Newtonian Scalar No-Go — Proof Chain Plan

## Purpose

This proof group promotes the scalar-completion failure into an explicit negative-result chain.

The scalar boundary/projection ladder successfully captures the Newtonian trace/monopole sector:

```text
scalar admissibility -> finite flux -> 1/r potential -> Newtonian force
```

The question here is different:

```text
Can a scalar-only nonlinear completion reproduce the post-Newtonian tensor structure of GR?
```

The target answer is conservative:

```text
A scalar-only completion can reproduce a Newtonian scalar potential and some isotropic metric response, but it cannot reproduce the full GR post-Newtonian structure without importing tensor/shear/vector/TT degrees of freedom or hard-coding them into the metric ansatz.
```

## What this group closes

This group closes several algebraic obstructions:

1. A scalar field supplies one local channel; a 3D spatial metric perturbation has six symmetric components and five traceless/shear components beyond trace.
2. A scalar conformal metric perturbation is trace-only and has zero traceless shear.
3. The most general isotropic static scalar metric ansatz has two scalar potentials, which can match the PPN parameter `gamma` only by choosing a second function; this is extra structure relative to one scalar potential.
4. A single scalar cannot encode vector gravitomagnetic components `g_0i`.
5. A scalar cannot encode TT radiative polarizations.
6. Scalar polynomial nonlinearities produce powers of the scalar potential, but GR post-Newtonian expansion contains tensor contractions and source-motion structures not determined by scalar powers alone.

## What remains imported

This group does not prove full GR. It assumes the comparison target is the standard weak-field/post-Newtonian tensor structure. It also does not prove the directional/tensor branch; it only proves that the scalar-only branch is insufficient.

## Batches

### Batch A — Scalar branch capability

Show that the scalar branch can reproduce Newtonian Poisson/Gauss/1/r behavior.

### Batch B — Tensor data missing from scalar branch

Count components and show trace-only/conformal response cannot recover shear, off-diagonal, vector, or TT data.

### Batch C — PPN obstruction

Show that the standard isotropic weak-field metric needs independent coefficients/potentials for `g_00` and `g_ij`. A one-scalar conformal ansatz fixes the wrong relation/sign unless extra structure is added.

### Batch D — Nonlinear scalar powers are not tensor structure

Show that scalar polynomial powers cannot generate independent tensor invariants, vector gravitomagnetic data, or TT modes.

### Batch E — Conclusion

Record the negative result: scalar-only nonlinear completion is a Newtonian trace-sector completion, not a GR post-Newtonian completion.
