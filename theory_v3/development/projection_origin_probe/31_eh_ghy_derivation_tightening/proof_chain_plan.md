# EH/GHY Derivation Tightening Proof Chain Plan

## Purpose

This folder tightens the action-side bridge after the scalar boundary ledger,
Weyl/tensor origin gate, boundary symplectic closure, radiative memory gate,
and nonlinear constraint-closure folders.

It does **not** claim to derive Einstein gravity from the original projection
coefficient `r_k`. It asks a narrower question:

```text
Given metric interval structure, diffeomorphism/relabeling invariance,
second-order locality, boundary differentiability, and no extra un-routed
fields, why is the EH+GHY branch the minimal local action closure?
```

## Strategy

The chain separates four layers:

1. metric-volume and inverse-metric variation bookkeeping;
2. second-derivative variation: bulk term plus boundary derivative leakage;
3. GHY/Hamiltonian boundary cancellation and differentiability;
4. action-class selection: Lovelock minimality, curvature-squared routing,
   Lambda allowance, and dependency ledger.

## Intended conclusion

```text
metric + diffeomorphism + second-order locality + boundary differentiability
+ no extra un-routed fields + D=4
    -> EH+GHY is the minimal local metric action branch.
```

This is an action-side closure result under assumptions, not a derivation of
EH+GHY from `r_k` alone.
