# Quadratic Response Selector: Provisional Proof Chain

## Folder Role

This folder is the next selector-level proof chain after:

```text
vacuum_interval_directional_probe_origin
torsion_defect_exclusion
vacuum_dimension_selector
```

The previous selector folders established conditional gates:

```text
directional interval probes can recover symmetric tensor data by polarization;
Levi-Civita transport is the minimal calibration-coherent affine branch;
D=4 is the minimal propagating Einstein branch under accepted spin-2/action gates.
```

This folder tests the earlier and more load-bearing selector:

```text
Why is the local interval-response primitive quadratic?
```

Equivalently:

```text
When is the response functional Q_p(v) genuinely metric-like rather than
Finsler-like or generally direction/scale dependent?
```

The purpose is not to re-derive the original scalar projection ratio. The
purpose is to isolate the exact condition under which directional interval
response collapses to a symmetric bilinear metric tensor.

## Core Risk

The downstream geometric tower uses:

```text
Q_p(v) = g_p(v,v)
```

or, perturbatively,

```text
Q_p(v) = eta(v,v) + h_p(v,v).
```

That requires a parallelogram/quadratic property. If the response is instead
nonquadratic, then polarization does not produce a direction-independent
bilinear form. The natural branch may be Finsler-like, medium-like, or an
explicit higher-order response sector.

This folder should therefore separate:

```text
exact quadratic metric response
```

from:

```text
local Hessian approximation near a stationary state
```

and from:

```text
nonquadratic directional response that must be routed outside the metric branch.
```

## Desired Endpoint

Strong closeout:

```text
local reciprocal interval response is forced by the vacuum ontology to obey the
parallelogram law exactly, so the metric branch is not an imported assumption.
```

Honest weaker closeout:

```text
this folder proves the exact algebraic gate: if Q obeys the parallelogram law,
polarization recovers a symmetric bilinear metric; if Q contains nonquadratic
terms, the metric branch fails or becomes only a Hessian approximation. The
parent ontology must still justify exact quadraticity or explicitly route the
nonquadratic branch.
```

Rejected move:

```text
assume Q is quadratic because GR is the target.
```

That would smuggle the metric branch.
