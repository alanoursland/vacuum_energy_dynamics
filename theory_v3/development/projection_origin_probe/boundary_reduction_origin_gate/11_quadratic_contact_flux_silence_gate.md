# 11. Quadratic contact flux-silence gate

Near a support boundary, write `s = R-r`. If `phi ~ s^m`, then radial flux is proportional to `-d phi/ds`.

- m=1: boundary derivative limit = -C
- m=2: boundary derivative limit = 0
- m=3: boundary derivative limit = 0
- m=4: boundary derivative limit = 0

Linear contact leaves finite boundary derivative; quadratic and higher contact make the derivative vanish.

Conclusion: quadratic contact is a standard boundary-regularity requirement for flux silence. It is not itself the underlying source dynamics.
