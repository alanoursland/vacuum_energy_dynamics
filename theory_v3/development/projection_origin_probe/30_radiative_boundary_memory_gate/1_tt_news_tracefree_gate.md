# TT news trace-free gate

A two-polarization boundary news tensor may be written in a transverse two-plane as

```text
N_AB = [[N_+, N_x], [N_x, -N_+]].
```

The script verifies

```text
tr(N)=0,
N_AB N^AB = 2(N_+^2 + N_x^2).
```

Thus scalar trace data can vanish while radiative tensor data is nonzero.
