# 10. Ricci-flat wave / Weyl-nonzero witness

Vacuum gravitational radiation is Ricci-flat but can carry nonzero Weyl curvature. As a minimal algebraic witness, set

```text
|Ricci|^2 = 0,
|Weyl|^2 = C^2.
```

Validated checks:

```text
Ricci norm = 0
Weyl norm != 0
```

Result: free curvature data lives outside the scalar Ricci/source ledger.
