# 6. Electric/magnetic Weyl exchange toy

A radiative Weyl-like pair can be modeled by

```text
E = A cos(kz - wt),
B = A sin(kz - wt).
```

Validated transport identities:

```text
∂_t E + w B = 0
∂_t B - w E = 0
E^2 + B^2 = A^2
```

Result: free tensor curvature data has transport structure; it is not just static scalar boundary charge.
