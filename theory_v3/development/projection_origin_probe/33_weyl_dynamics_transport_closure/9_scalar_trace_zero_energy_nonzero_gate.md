# 9. Scalar trace zero but energy nonzero gate

A general TT polarization matrix

```text
[[A, B, 0],
 [B,-A, 0],
 [0, 0, 0]]
```

has zero scalar trace but nonzero quadratic amplitude.

Validated checks:

```text
tr(h) = 0
h_ij h_ij = 2A^2 + 2B^2
```

Result: scalar trace blindness does not mean no physical radiative content.
