# 2. TT plane wave transverse gate

For a wave traveling in the `z` direction, transversality means

```text
∂_i h_ij = 0.
```

Using a plus-polarized wave with only `xx` and `yy` components and no `zj` components, the divergence reduces to `∂_z h_zj = 0`.

Validated result:

```text
∂_i h_ij = 0.
```

Thus the same mode is both trace-free and transverse.
