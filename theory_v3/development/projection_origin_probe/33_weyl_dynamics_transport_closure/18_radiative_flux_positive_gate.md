# 18. Radiative flux positive gate

With plus/cross news components `N_+` and `N_x`, the quadratic flux witness is

```text
F = N_+^2 + N_x^2.
```

Validated checks:

```text
F != 0 generically
F = 0 when N_+ = N_x = 0
```

Result: radiative flux is carried by tensor news, not scalar monopole charge.
