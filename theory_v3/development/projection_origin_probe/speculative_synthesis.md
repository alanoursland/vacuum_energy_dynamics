# Projection Origin Probe: Speculative Synthesis

This document synthesizes the projection-origin probe results in this folder.
It is intentionally speculative, but the speculation is constrained by one
standard:

```text
Resemblance is not derivation.
```

The central finding is:

```text
The projection hierarchy has a real integration-by-parts/operator origin,
but that origin belongs to a broader primitive-power family. The observed
m = 2 case is structurally attractive, but it is not independently selected
unless an external measure, boundary, or variational principle forces it.
```

That is the through-line of the probe. The hierarchy is not formal noise, but
it is also not yet a physical result.

## 1. The Object and the Mystery

The central projection form is

```text
A[k,j] = 2 integral_0^1 psi_k(x) phi_j(x) (1 - x^2)^4 dx
```

with

```text
phi_j(x) = x^(2j)
psi_k(x) = x^(2k) - ((2k - 1)/(2k + 3)) x^(2k - 2)
w(x) = (1 - x^2)^4.
```

Write

```text
a = 1 - x^2.
```

Then

```text
w = a^4.
```

The row-test function can be written

```text
psi_k(x) = x^(2k - 2)(x^2 - r_k)
r_k = (2k - 1)/(2k + 3).
```

Each `psi_k` changes sign at

```text
x_k = sqrt((2k - 1)/(2k + 3)).
```

That sign change is the first important diagnostic. It blocks the naive
positive-Gram interpretation. The hierarchy is not naturally read as a
positive energy matrix or Hessian. It behaves more like a compatibility,
balance, or admissibility projection.

The ratio

```text
(2k - 1)/(2k + 3)
```

is the mystery. The numerator and denominator are too structured to treat as
arbitrary, but the sign-changing rows make the usual positive-weight
orthogonality story suspect.

## 2. The IBP Identity Resolves the Ratio

The key formal result is the derivative identity

```text
d/dx [x^(2k - 1)(1 - x^2)^2]
  = -(2k + 3)(1 - x^2) psi_k(x).
```

Equivalently,

```text
psi_k(x)
  = -1 / ((2k + 3)(1 - x^2))
    d/dx [x^(2k - 1)(1 - x^2)^2].
```

Thus `psi_k` is generated by the boundary-vanishing primitive

```text
G_k(x) = x^(2k - 1)a^2.
```

The ratio is no longer unexplained. It is the product-rule residue produced by
differentiating `x^(2k - 1)a^2`. The `(2k - 1)` comes from the monomial
derivative, and the `(2k + 3)` appears after collecting the two product-rule
terms.

This also explains a negative result. Under the auxiliary weight `a`,
the ratio

```text
integral x^(2k) a dx / integral x^(2k - 2) a dx
```

is exactly

```text
(2k - 1)/(2k + 3).
```

But under the actual projection weight `a^4`, the analogous same-weight ratio
is

```text
(2k - 1)/(2k + 9).
```

So the observed `psi_k` is not simply the result of same-weight moment
orthogonality under `w = a^4`. Its origin is the derivative primitive and the
integration-by-parts pullback.

## 3. The Hidden Pullback Operator

Start with the pairing

```text
I_k[f] = integral_0^1 psi_k(x) f(x) a^4 dx.
```

Using the primitive identity and integrating by parts gives

```text
<psi_k, f>_w
  = 1/(2k + 3) <x^(2k - 1), L[f]>_w,
```

where

```text
L[f] = a f' - 6xf.
```

The same operator has the divergence form

```text
L[f] = a^(-2) d/dx [a^3 f].
```

This is the second main result. The even projection rows are equivalent to odd
moment tests of the transformed profile `L[f]`:

```text
left side:
  even row tests against f

right side:
  odd moments of L[f]
```

The parity flip is not decorative. The basis functions are even:

```text
phi_j(x) = x^(2j),
```

but the pullback exposes odd powers:

```text
x^(2k - 1).
```

That is exactly what a first-order operator should do. It suggests that the
projection is more naturally about flux, compatibility, or balance than about
positive magnitude.

The divergence form also identifies a useful transformed variable:

```text
u = a^3 f.
```

Then

```text
L[f] = a^(-2) u'.
```

Consequently the quadratic flux term simplifies exactly:

```text
<L[f], L[f]>_w
  = integral_0^1 (a^(-2)u')^2 a^4 dx
  = integral_0^1 (u')^2 dx.
```

So in the variable `u = a^3 f`, the formal energy associated with `L` is
ordinary Dirichlet energy. This does not derive a physical functional, but it
is one of the cleanest mathematical simplifications in the probe.

The same exponent structure appears in the source-family diagnostic. For
formal source probes

```text
S_(p,q)(x) = x^(2q)(1 - x^2)^p,
```

the two parameters probe the endpoints separately:

```text
p: vanishing order at x = 1
q: vanishing order at x = 0.
```

For this family, the sign structure of the projection response is controlled,
up to positive factors, by

```text
-(2kp + 6k - p - 4q - 3).
```

The same coefficient `6` appears here and in

```text
L[f] = a f' - 6xf.
```

This does not turn the source family into a physical source law. It means the
endpoint diagnostic is governed by the same exponent bookkeeping that produced
the pullback operator. The source probes are therefore useful as domain and
endpoint tests, not as an interpretation by themselves.

## 4. The Adjoint Structure and Parent-Operator Candidate

Under the projection weight

```text
w = a^4,
```

the weighted adjoint is

```text
L_w^*[g] = -a g' + 4xg.
```

The integration-by-parts boundary term is

```text
[a^5 f g]_0^1.
```

For regular functions it is clean at `x = 1`, and for odd monomial tests
`g = x^(2k - 1)` it is clean at `x = 0`.

The derived second-order compositions are

```text
L_w^*L[f]
  = -a^2 f'' + 12xa f' + (6 - 30x^2)f
```

and

```text
L L_w^*[g]
  = -a^2 g'' + 12xa g' + (4 - 28x^2)g.
```

These are natural parent-operator candidates, but they are not yet parent
equations. They should be kept as tools unless a concrete variational,
Sturm-Liouville, boundary-domain, or self-adjointness problem appears.

One useful diagnostic is the zeroth-order residue:

```text
(6 - 30x^2) - (4 - 28x^2)
  = 2 - 2x^2
  = 2a.
```

The operator pair has a clean commutator-like residue proportional to the base
factor `a`. This is structurally suggestive, but it is not an external
derivation.

The projection weight also sits near a standard orthogonal-polynomial setting.
On the full interval `[-1,1]`,

```text
(1 - x^2)^4
```

is a Gegenbauer weight with

```text
lambda = 9/2,
```

because Gegenbauer weights have the form

```text
(1 - x^2)^(lambda - 1/2).
```

Equivalently, it is a symmetric Jacobi weight. The present construction uses
the even half of this setting on `[0,1]`. This context is useful, but it does
not derive the hierarchy, and it does not select `m = 2`.

## 5. The Primitive-Power Family Exposes the Artifact Risk

The decisive stress test is to replace the observed primitive with a family:

```text
G_(k,m)(x) = x^(2k - 1)a^m.
```

Then

```text
G_(k,m)' = -(2k + 2m - 1)a^(m - 1) psi_(k,m)(x),
```

where

```text
psi_(k,m)(x)
  = x^(2k) - ((2k - 1)/(2k + 2m - 1)) x^(2k - 2).
```

The observed hierarchy is the `m = 2` member because

```text
2k + 2m - 1 = 2k + 3
```

forces

```text
m = 2.
```

For the family, the pullback operator becomes

```text
L_m[f] = a f' - 2(5 - m)xf.
```

The observed operator is

```text
L_2[f] = a f' - 6xf.
```

For the regular range `1 <= m <= 5`, the same integration-by-parts machinery
works. The boundary term remains

```text
[a^5 f g]_0^1.
```

This is the main artifact warning:

```text
The existence of an IBP construction does not uniquely select m = 2.
```

The original result is real, but the construction is not unique. The observed
case is one member of a structured family.

## 6. What Still Selects `m = 2`

Several candidate selectors were tested.

First, the observed ratio selects `m = 2`, but this is not independent. The
ratio is how the observed member is identified.

Second, the weighted-adjoint family has real structure. Under `w = a^4`,

```text
L_m^*[g] = -a g' + 2m xg,
```

and

```text
L_m^* = -L_(5 - m).
```

So the family closes under the weighted adjoint:

```text
m = 1 <-> m = 4
m = 2 <-> m = 3
m = 3 <-> m = 2
m = 4 <-> m = 1
```

The fixed skew-adjoint point would be

```text
m = 5/2,
```

not an integer primitive power. Therefore adjoint closure gives structure, but
not a unique selection of `m = 2`.

Third, the compactified radial measure test fails as a selector. For

```text
r = x / a^c,  c > 0,
```

an `n`-dimensional radial measure contributes

```text
a^(-cn - 1).
```

For positive `c` and positive `n`, this is a negative power of `a`. It cannot
naturally produce the positive projection weight

```text
a^4.
```

The standard case `c = 1/2` would formally require

```text
n = -10.
```

Thus ordinary compactified radial measure does not explain the weight and does
not select `m = 2`.

The remaining internal clue is the exponent ladder. For general `m`, the
exponents are

```text
primitive:   a^m
flux:        a^(5 - m)
weight:      a^4
boundary:    a^5
```

For the observed case `m = 2`, this becomes

```text
primitive:   a^2
flux:        a^3
weight:      a^4
boundary:    a^5
```

or

```text
a^2 -> a^3 -> a^4 -> a^5.
```

This is the cleanest internal clue, but it is still conditional. It selects
`m = 2` only if the ordered adjacent ladder is treated as a principle.

The generalized version makes this sharper. If

```text
w = a^beta,
```

then the analogous roles are

```text
primitive:   a^m
flux:        a^(beta + 1 - m)
weight:      a^beta
boundary:    a^(beta + 1)
```

Requiring a strict adjacent ordered ladder forces

```text
m = beta / 2.
```

Equivalently:

```text
primitive power = weight power / 2.
```

For `beta = 4`, this gives `m = 2`. The half-weight relation is a compact
mathematical summary of the ladder clue. It becomes a derivation only if an
external principle first explains both the weight `a^4` and the ordered ladder
requirement.

## 7. The Cleanest Reopen Condition: A Variational Object

The strongest speculative anchor is a variational one:

```text
If a concrete energy functional produces L_w^*L as its Euler-Lagrange
operator, the projection hierarchy may become a Galerkin or moment projection
of that variational problem.
```

A candidate shape is

```text
E[f] = 1/2 <L[f], L[f]>_w - <S, f>_w.
```

Then, for admissible variations `eta`,

```text
delta E[f; eta]
  = <L[f], L[eta]>_w - <S, eta>_w.
```

Using the weighted adjoint and assuming the boundary terms vanish on the
chosen domain,

```text
delta E[f; eta]
  = <L_w^*L[f] - S, eta>_w.
```

The formal Euler-Lagrange equation is therefore

```text
L_w^*L[f] = S.
```

In the transformed variable

```text
u = a^3 f,
```

the quadratic part becomes

```text
1/2 <L[f], L[f]>_w
  = 1/2 integral_0^1 (u')^2 dx.
```

So the proposed variational target is, after a change of variable, ordinary
Dirichlet energy plus a transformed source coupling. That is mathematically
clean. It is not yet physically derived.

If such a functional were independently supplied, the hierarchy could be read
as a finite-dimensional Galerkin or moment projection of the equation

```text
L_w^*L[f] = S.
```

The projection matrix would no longer be merely formal. It would be a
stiffness-like matrix associated with the energy problem.

To promote this from a speculative target to a result, one would need an
independent derivation of:

- the energy form `1/2 <L[f], L[f]>_w`;
- the weight `w = a^4`;
- the source pairing `<S, f>_w`;
- the allowed variation space;
- the boundary conditions that make the adjoint calculation clean;
- the reason this energy selects `L = L_2` rather than another `L_m`.

The zero-source case also needs care. Since

```text
L[f] = a^(-2) d/dx [a^3 f],
```

the condition

```text
L[f] = 0
```

implies

```text
d/dx [a^3 f] = 0.
```

Therefore

```text
a^3 f = constant
```

and

```text
f = constant / a^3.
```

This is singular at `x = 1` unless the constant is zero. Under regular
boundary conditions, the only source-free zero-energy profile in the `f`
variable is

```text
f = 0.
```

In the transformed variable `u`, the zero-energy profiles are constants, but
regularity in `f = u/a^3` forces that constant to vanish unless the domain or
baseline is changed.

Thus the variational picture does not by itself produce a nonzero trivial
profile. A nonzero baseline would require a different variable, a shifted
energy, a constraint, or a separately specified background state.

## 8. Final Status

The probe establishes real structure:

```text
psi_k:
  derived from a boundary-vanishing primitive

r_k:
  derived from product-rule algebra once m = 2 is chosen

L[f]:
  derived as the weighted-divergence pullback

L_w^*:
  derived under the projection weight

L_w^*L:
  available as a parent-operator candidate

u = a^3 f:
  turns the quadratic <Lf,Lf>_w term into ordinary Dirichlet energy
```

It also establishes the artifact risk:

```text
primitive family:
  every regular m gives an IBP construction

m = 2:
  selected by the observed ratio
  suggested by the adjacent exponent ladder
  not independently selected

compactified radial measure:
  does not explain a^4

orthogonal-polynomial neighborhood:
  contextual, not selective
```

The resulting status is:

```text
projection hierarchy:
  formal but structured

physical interpretation:
  not derived

reopen condition:
  provide a concrete external object that forces the weight, operator,
  domain, source pairing, or variational structure
```

Until then, the correct state is containment rather than dismissal or
interpretive expansion.
