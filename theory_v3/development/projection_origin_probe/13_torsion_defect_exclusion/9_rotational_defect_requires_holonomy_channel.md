# Torsion Defect Exclusion 9: Rotational Defect Requires Holonomy Channel

## Purpose

This proof checks the route for rotational/closure-failure torsion sources.

A rotational defect requires antisymmetric holonomy data. Symmetric deformation
data is not enough.

## Validated Checks

- antisymmetric part of a local comparison map carries the rotation: passed
- symmetric part has no antisymmetric component: passed
- if the holonomy/closure residual vanishes, the defect source vanishes: passed

## Model

For a local comparison map:

```text
M = [[a,b],[c,d]],
```

the antisymmetric component is:

```text
A = (M-M^T)/2.
```

The rotation/closure witness is:

```text
theta = (c-b)/2.
```

A defect source of the form:

```text
J_defect = kappa theta
```

vanishes when:

```text
b = c.
```

## Interpretation

Rotational defect torsion is not generated by symmetric interval data. It
requires an independent holonomy, closure-failure, or orientation-rotation
channel.
