# Candidate R0 Embedding Condition

If the GR scalar boundary ledger is embedded using the same projection chart

```text
y=x^2,   C_0[P]=∫P(y)(1-y)y^(-1/2)dy,
```

then by construction the GR ledger lands in the observed `R=0` projection class.

This is a valid compatibility embedding, not an invariant derivation of `R=0` from GR alone.
