# Different Embedding R Witness

If an additional endpoint contact factor `(1-y)^R` is inserted into the test pairing, the measured ratio changes to

```text
r_(R,k) = (2k - 1)/(2k + 2R + 3).
```

For example, `R=1` changes `r_(1,1)` from `1/5` to `1/7`.

This is a direct witness that the contact class is projection-embedding data.
