# 1. GR Branch Scope

## Claim

The GR branch is now a conditional reconstruction, not a derivation from the
original scalar projection coefficient alone.

The branch begins only after the metric gate is imposed:

```text
exact quadratic response / epsilon = 0
```

Once that branch is selected, the accumulated proof folders establish that the
standard GR structures appear in the expected places:

```text
scalar trace sector,
Weyl/TT tensor sector,
radiative boundary phase space,
metric stress route,
Hamiltonian constraint closure,
EH+GHY action completion.
```

## Not claimed

This capstone does not claim:

```text
r_k derives GR.
```

It also does not claim:

```text
the vacuum ontology has forced epsilon = 0.
```

Those are different questions.

## Status

The GR branch is closed as a conditional reconstruction under the metric,
diffeomorphism, locality, boundary, and matter-action assumptions already
identified in the proof chain.
