# GR Branch Closure Capstone — Plan

## Purpose

This folder is a capstone for the GR reconstruction branch. It is not a new
SymPy proof folder. The prior proof folders already validate the local algebra,
component counts, boundary ledgers, TT witnesses, action boundary mechanics, and
no-go results used here.

The purpose here is to declare exactly what is now closed in the `epsilon = 0`
metric branch, what assumptions remain imported, and what is no longer part of
GR-closure work.

## Core distinction

```text
GR branch closure != vacuum ontology closure.
```

The GR branch is the conditional reconstruction obtained after the metric gate
has been imposed:

```text
exact quadratic response / epsilon = 0
  -> metric interval structure
  -> Levi-Civita calibration branch
  -> scalar Newtonian boundary sector
  -> Weyl/TT tensor sector
  -> radiative boundary symplectic ledger
  -> stress-tensor matter route, if matter action exists
  -> nonlinear constraint closure pressure
  -> EH+GHY minimal local action branch.
```

The vacuum ontology closure is a separate frontier:

```text
Does the underlying response functional force epsilon = 0,
or does it leave a nonmetric/Finsler/constitutive residual?
```

## Capstone standard

The GR branch can be called done when:

1. The scalar trace/monopole sector is placed correctly.
2. The tensor/Weyl/TT sector is separated from scalar boundary data.
3. Radiative news, memory, and symplectic flux are routed to tensor boundary
   phase space rather than scalar charge.
4. Matter coupling is conditionally closed through metric stress variation.
5. Nonlinear constraint closure is conditionally closed under the usual metric,
   diffeomorphism, locality, and no-extra-field assumptions.
6. EH+GHY is identified as the minimal D=4 metric action branch under the same
   assumptions.
7. The remaining open question is explicitly moved out of GR reconstruction and
   into the ontology/epsilon frontier.

## Expected conclusion

```text
The project has completed a conditional GR reconstruction branch.
It has not completed the vacuum ontology derivation unless epsilon is computed
or otherwise forced to zero.
```
