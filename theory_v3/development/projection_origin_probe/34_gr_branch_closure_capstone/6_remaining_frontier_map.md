# 6. Remaining Frontier Map

The remaining frontiers are no longer GR-sector closure questions. They are
ontology or beyond-GR questions.

## Frontier 1: epsilon computation

```text
Does the vacuum configuration functional force epsilon = 0?
```

Outcomes:

```text
epsilon = 0     -> exact GR branch recovered.
epsilon != 0    -> nonmetric/Finsler/constitutive deviation branch.
epsilon undefined -> vacuum ontology under-specified.
```

## Frontier 2: matter microstructure

```text
Can matter existence, species, mass, charge, spin, localization, and quantum
structure be derived?
```

Current status: open.

## Frontier 3: Lambda mechanism

```text
What selects or relaxes the global vacuum baseline?
```

Current status: branch-classified, not dynamically derived.

## Frontier 4: quantum structure

```text
Does the relational/vacuum ontology generate quantum state space, amplitudes,
commutators, path integral, or particle spectra?
```

Current status: absent.

## Frontier 5: possible deviations

```text
If epsilon != 0 or matter/vacuum routing differs, what observables change?
```

This is where novel predictions would live.

## Recommendation

Do not continue writing GR-reconstruction folders unless a specific new
technical gap is identified. Future work should move to epsilon, matter,
Lambda, quantum, or deviation-analysis folders.
