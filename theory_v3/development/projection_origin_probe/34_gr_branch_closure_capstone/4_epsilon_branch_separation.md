# 4. Epsilon Branch Separation

## Core residual

Let the local directional response be schematically written as

```text
Q(v) = g_ab v^a v^b + epsilon * F_4(v) + higher residuals.
```

The GR branch corresponds to

```text
epsilon = 0.
```

The nonmetric branch corresponds to

```text
epsilon != 0,
```

or more generally to directional response that cannot be represented by a
single symmetric bilinear metric.

## What has been shown

The proof chain has shown that if the physical consistency gates require exact
parallelogram response, degree-two homogeneity, scale-independent calibration,
universal null-cone ownership, polarization path-independence, and constant
matrix representability, then the tested quartic/Finsler residual is forced
out of the metric branch.

## What has not been shown

The chain has not computed `epsilon` from a fully specified underlying vacuum
configuration functional.

Therefore the project has not yet proven:

```text
vacuum ontology -> epsilon = 0.
```

## Consequence

GR closure and ontology closure are now cleanly separated.

```text
GR closure: done conditionally at epsilon = 0.
Ontology closure: open until epsilon is computed or forced to zero.
```
