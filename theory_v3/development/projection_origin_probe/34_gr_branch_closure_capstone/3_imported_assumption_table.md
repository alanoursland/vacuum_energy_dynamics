# 3. Imported Assumption Table

## Assumptions still imported into the GR branch

| Assumption | Role | Current status |
|---|---|---|
| Exact quadratic response | Produces pseudo-Riemannian metric structure by polarization. | Not derived from the vacuum functional; tested as a selector gate. |
| `epsilon = 0` | Removes nonmetric/Finsler/constitutive residuals from interval response. | Not computed from first principles. |
| Shared interval ownership | Makes all matter actions use the same metric interval. | Conditionally justified operationally; not microscopically derived. |
| Matter action existence | Supplies stress tensor source route. | Open. |
| Diffeomorphism/relabeling redundancy | Needed for GR gauge structure and conservation identities. | Strongly motivated by relational comparison; treated as branch assumption. |
| Second-order locality | Selects EH over higher-curvature branches in D=4. | Action-class assumption. |
| No extra un-routed fields | Excludes torsion, nonmetricity, massive modes, scalar/vector residuals. | Routing rule, not an absolute impossibility theorem. |
| Lorentzian/time channel | Supports hyperbolic propagation and TT radiation. | Conditionally selected; not derived from scalar ladder alone. |
| Global Lambda branch | Determines asymptotic vacuum class. | Not fixed locally. |
| Quantum structure | Would supply Hilbert space/path integral/commutators/quanta. | Not present in current classical chain. |

## Meaning

The GR branch is not empty because these assumptions are not arbitrary inside
the branch; many have been narrowed by gates and no-go results. But the branch
is still conditional because these assumptions are not all derived from the
vacuum ontology itself.
