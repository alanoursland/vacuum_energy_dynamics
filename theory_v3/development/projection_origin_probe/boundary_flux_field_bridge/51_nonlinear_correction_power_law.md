# Boundary Flux Field Bridge 51: Nonlinear Correction Power Law

## Purpose

This report classifies the first weak-field correction generated by nonlinear
radial strain densities:

```text
Phi_p(z) = 1/2 z + alpha/(2p) z^p.
```

The Dirichlet model is recovered at `alpha=0`.

## Validated Checks

- Phi_p correction law verified for p=2..7: passed

## Flux Law

Let:

```text
y = -u' > 0.
```

For `Phi_p`, the radial flux law is:

```text
Q = 4*pi*r^2 [y + alpha y^(2p-1)].
```

With:

```text
y0 = Q/(4*pi*r^2),
```

the first-order perturbative solution is:

```text
y = y0 - alpha y0^(2p-1) + O(alpha^2).
```

Integrating from infinity gives:

```text
u(r)
  =
  Q/(4*pi*r)
  - alpha (Q/(4*pi))^(2p-1) / [(4p-3) r^(4p-3)]
  + O(alpha^2).
```

## Checked Cases


## Interpretation

The first nonlinear correction has a definite power:

```text
r^(-(4p-3)).
```

For `p=2`, this recovers proof `44`:

```text
r^-5.
```

This gives a compact classifier for candidate nonlinear scalar vacuum energies.
Each nonlinear strain power predicts a different far-field correction hierarchy.
