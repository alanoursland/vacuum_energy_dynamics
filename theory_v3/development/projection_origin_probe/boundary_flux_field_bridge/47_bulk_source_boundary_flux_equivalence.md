# Boundary Flux Field Bridge 47: Bulk Source and Boundary Flux Equivalence

## Purpose

This report validates that, in the spherical exterior problem, a compact bulk
source and an enclosing boundary flux are equivalent outside the source region.

This is the standard shell/Gauss-law behavior, written in the bookkeeping used
by this branch.

## Validated Checks

- polynomial bulk source total charge: passed
- exterior field is harmonic: passed
- exterior flux equals enclosed source: passed
- exterior flux is conserved: passed
- boundary flux reproduces bulk source total: passed
- boundary potential from enclosed source: passed

## Representative Bulk Source

Inside radius `R`, take:

```text
rho(r) = rho0 + rho2 r^2 + rho4 r^4.
```

The total enclosed source is:

```text
Q = 4*pi*(rho0 R^3/3 + rho2 R^5/5 + rho4 R^7/7).
```

## Exterior Field

For `r > R`, the exterior solution is:

```text
u(r) = Q/(4*pi*r).
```

SymPy verifies:

```text
Delta u = 0
-4*pi*r^2*u'(r) = Q.
```

## Boundary Flux Equivalence

The same exterior field is generated by imposing the boundary flux condition:

```text
-4*pi*R^2*u'(R) = Q.
```

Therefore:

```text
compact spherical bulk source
```

and:

```text
enclosing boundary flux
```

are equivalent for the exterior field.

## Interpretation

This supports the boundary-source reading without denying ordinary bulk-source
language. In the exterior vacuum problem, only the enclosed flux/charge matters.
