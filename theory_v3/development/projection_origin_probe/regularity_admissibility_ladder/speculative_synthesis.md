# Projection Origin Probe: Speculative Synthesis

This document synthesizes the projection-origin probe results in this folder.
It is intentionally speculative, but the speculation is constrained by one
standard:

```text
Resemblance is not derivation.
```

The central finding is:

```text
The projection hierarchy has a real integration-by-parts/operator origin,
but that origin belongs to a broader primitive-power family. The observed
m = 2 case is structurally attractive, but it is not independently selected
unless an external measure, boundary, or variational principle forces it.
```

That is the through-line of the probe. The hierarchy is not formal noise, but
it is also not yet a physical result.

## 1. The Object and the Mystery

The central projection form is

```text
A[k,j] = 2 integral_0^1 psi_k(x) phi_j(x) (1 - x^2)^4 dx
```

with

```text
phi_j(x) = x^(2j)
psi_k(x) = x^(2k) - ((2k - 1)/(2k + 3)) x^(2k - 2)
w(x) = (1 - x^2)^4.
```

Write

```text
a = 1 - x^2.
```

Then

```text
w = a^4.
```

The row-test function can be written

```text
psi_k(x) = x^(2k - 2)(x^2 - r_k)
r_k = (2k - 1)/(2k + 3).
```

Each `psi_k` changes sign at

```text
x_k = sqrt((2k - 1)/(2k + 3)).
```

That sign change is the first important diagnostic. It blocks the naive
positive-Gram interpretation. The hierarchy is not naturally read as a
positive energy matrix or Hessian. It behaves more like a compatibility,
balance, or admissibility projection.

The ratio

```text
(2k - 1)/(2k + 3)
```

is the mystery. The numerator and denominator are too structured to treat as
arbitrary, but the sign-changing rows make the usual positive-weight
orthogonality story suspect.

## 2. The IBP Identity Resolves the Ratio

The key formal result is the derivative identity

```text
d/dx [x^(2k - 1)(1 - x^2)^2]
  = -(2k + 3)(1 - x^2) psi_k(x).
```

Equivalently,

```text
psi_k(x)
  = -1 / ((2k + 3)(1 - x^2))
    d/dx [x^(2k - 1)(1 - x^2)^2].
```

Thus `psi_k` is generated by the boundary-vanishing primitive

```text
G_k(x) = x^(2k - 1)a^2.
```

The ratio is no longer unexplained. It is the product-rule residue produced by
differentiating `x^(2k - 1)a^2`. The `(2k - 1)` comes from the monomial
derivative, and the `(2k + 3)` appears after collecting the two product-rule
terms.

This also explains a negative result. Under the auxiliary weight `a`,
the ratio

```text
integral x^(2k) a dx / integral x^(2k - 2) a dx
```

is exactly

```text
(2k - 1)/(2k + 3).
```

But under the actual projection weight `a^4`, the analogous same-weight ratio
is

```text
(2k - 1)/(2k + 9).
```

So the observed `psi_k` is not simply the result of same-weight moment
orthogonality under `w = a^4`. Its origin is the derivative primitive and the
integration-by-parts pullback.

## 3. The Hidden Pullback Operator

Start with the pairing

```text
I_k[f] = integral_0^1 psi_k(x) f(x) a^4 dx.
```

Using the primitive identity and integrating by parts gives

```text
<psi_k, f>_w
  = 1/(2k + 3) <x^(2k - 1), L[f]>_w,
```

where

```text
L[f] = a f' - 6xf.
```

The same operator has the divergence form

```text
L[f] = a^(-2) d/dx [a^3 f].
```

This is the second main result. The even projection rows are equivalent to odd
moment tests of the transformed profile `L[f]`:

```text
left side:
  even row tests against f

right side:
  odd moments of L[f]
```

The parity flip is not decorative. The basis functions are even:

```text
phi_j(x) = x^(2j),
```

but the pullback exposes odd powers:

```text
x^(2k - 1).
```

That is exactly what a first-order operator should do. It suggests that the
projection is more naturally about flux, compatibility, or balance than about
positive magnitude.

The divergence form also identifies a useful transformed variable:

```text
u = a^3 f.
```

Then

```text
L[f] = a^(-2) u'.
```

Consequently the quadratic flux term simplifies exactly:

```text
<L[f], L[f]>_w
  = integral_0^1 (a^(-2)u')^2 a^4 dx
  = integral_0^1 (u')^2 dx.
```

So in the variable `u = a^3 f`, the formal energy associated with `L` is
ordinary Dirichlet energy. This does not derive a physical functional, but it
is one of the cleanest mathematical simplifications in the probe.

The same exponent structure appears in the source-family diagnostic. For
formal source probes

```text
S_(p,q)(x) = x^(2q)(1 - x^2)^p,
```

the two parameters probe the endpoints separately:

```text
p: vanishing order at x = 1
q: vanishing order at x = 0.
```

For this family, the sign structure of the projection response is controlled,
up to positive factors, by

```text
-(2kp + 6k - p - 4q - 3).
```

The same coefficient `6` appears here and in

```text
L[f] = a f' - 6xf.
```

This does not turn the source family into a physical source law. It means the
endpoint diagnostic is governed by the same exponent bookkeeping that produced
the pullback operator. The source probes are therefore useful as domain and
endpoint tests, not as an interpretation by themselves.

## 4. The Adjoint Structure and Parent-Operator Candidate

Under the projection weight

```text
w = a^4,
```

the weighted adjoint is

```text
L_w^*[g] = -a g' + 4xg.
```

The integration-by-parts boundary term is

```text
[a^5 f g]_0^1.
```

For regular functions it is clean at `x = 1`, and for odd monomial tests
`g = x^(2k - 1)` it is clean at `x = 0`.

The derived second-order compositions are

```text
L_w^*L[f]
  = -a^2 f'' + 12xa f' + (6 - 30x^2)f
```

and

```text
L L_w^*[g]
  = -a^2 g'' + 12xa g' + (4 - 28x^2)g.
```

These are natural parent-operator candidates, but they are not yet parent
equations. They should be kept as tools unless a concrete variational,
Sturm-Liouville, boundary-domain, or self-adjointness problem appears.

One useful diagnostic is the zeroth-order residue:

```text
(6 - 30x^2) - (4 - 28x^2)
  = 2 - 2x^2
  = 2a.
```

The operator pair has a clean commutator-like residue proportional to the base
factor `a`. This is structurally suggestive, but it is not an external
derivation.

The projection weight also sits near a standard orthogonal-polynomial setting.
On the full interval `[-1,1]`,

```text
(1 - x^2)^4
```

is a Gegenbauer weight with

```text
lambda = 9/2,
```

because Gegenbauer weights have the form

```text
(1 - x^2)^(lambda - 1/2).
```

Equivalently, it is a symmetric Jacobi weight. The present construction uses
the even half of this setting on `[0,1]`. This context is useful, but it does
not derive the hierarchy, and it does not select `m = 2`.

## 5. The Primitive-Power Family Exposes the Artifact Risk

The decisive stress test is to replace the observed primitive with a family:

```text
G_(k,m)(x) = x^(2k - 1)a^m.
```

Then

```text
G_(k,m)' = -(2k + 2m - 1)a^(m - 1) psi_(k,m)(x),
```

where

```text
psi_(k,m)(x)
  = x^(2k) - ((2k - 1)/(2k + 2m - 1)) x^(2k - 2).
```

The observed hierarchy is the `m = 2` member because

```text
2k + 2m - 1 = 2k + 3
```

forces

```text
m = 2.
```

For the family, the pullback operator becomes

```text
L_m[f] = a f' - 2(5 - m)xf.
```

The observed operator is

```text
L_2[f] = a f' - 6xf.
```

For the regular range `1 <= m <= 5`, the same integration-by-parts machinery
works. The boundary term remains

```text
[a^5 f g]_0^1.
```

This is the main artifact warning:

```text
The existence of an IBP construction does not uniquely select m = 2.
```

The original result is real, but the construction is not unique. The observed
case is one member of a structured family.

## 6. What Still Selects `m = 2`

Several candidate selectors were tested.

First, the observed ratio selects `m = 2`, but this is not independent. The
ratio is how the observed member is identified.

Second, the weighted-adjoint family has real structure. Under `w = a^4`,

```text
L_m^*[g] = -a g' + 2m xg,
```

and

```text
L_m^* = -L_(5 - m).
```

So the family closes under the weighted adjoint:

```text
m = 1 <-> m = 4
m = 2 <-> m = 3
m = 3 <-> m = 2
m = 4 <-> m = 1
```

The fixed skew-adjoint point would be

```text
m = 5/2,
```

not an integer primitive power. Therefore adjoint closure gives structure, but
not a unique selection of `m = 2`.

Third, the compactified radial measure test fails as a selector. For

```text
r = x / a^c,  c > 0,
```

an `n`-dimensional radial measure contributes

```text
a^(-cn - 1).
```

For positive `c` and positive `n`, this is a negative power of `a`. It cannot
naturally produce the positive projection weight

```text
a^4.
```

The standard case `c = 1/2` would formally require

```text
n = -10.
```

Thus ordinary compactified radial measure does not explain the weight and does
not select `m = 2`.

The remaining internal clue is the exponent ladder. For general `m`, the
exponents are

```text
primitive:   a^m
flux:        a^(5 - m)
weight:      a^4
boundary:    a^5
```

For the observed case `m = 2`, this becomes

```text
primitive:   a^2
flux:        a^3
weight:      a^4
boundary:    a^5
```

or

```text
a^2 -> a^3 -> a^4 -> a^5.
```

This is the cleanest internal clue, but it is still conditional. It selects
`m = 2` only if the ordered adjacent ladder is treated as a principle.

The generalized version makes this sharper. If

```text
w = a^beta,
```

then the analogous roles are

```text
primitive:   a^m
flux:        a^(beta + 1 - m)
weight:      a^beta
boundary:    a^(beta + 1)
```

Requiring a strict adjacent ordered ladder forces

```text
m = beta / 2.
```

This is the primitive/flux balance condition:

```text
primitive power = weight power / 2.
```

For the entire four-term chain to be consecutive,

```text
primitive -> flux -> weight -> boundary,
```

the second adjacency condition also requires

```text
m = 2.
```

Together these give

```text
beta = 4
m = 2.
```

So the half-weight relation is a compact mathematical summary of the balance
clue, while the fully adjacent four-term ladder is more restrictive: it
selects the observed `beta = 4`, `m = 2` case. This becomes a derivation only
if an external principle first explains why the ordered ladder should be
required.

## 7. The Cleanest Reopen Condition: A Variational Object

The strongest speculative anchor is a variational one:

```text
If a concrete energy functional produces L_w^*L as its Euler-Lagrange
operator, the projection hierarchy may become a Galerkin or moment projection
of that variational problem.
```

A candidate shape is

```text
E[f] = 1/2 <L[f], L[f]>_w - <S, f>_w.
```

Then, for admissible variations `eta`,

```text
delta E[f; eta]
  = <L[f], L[eta]>_w - <S, eta>_w.
```

Using the weighted adjoint and assuming the boundary terms vanish on the
chosen domain,

```text
delta E[f; eta]
  = <L_w^*L[f] - S, eta>_w.
```

The formal Euler-Lagrange equation is therefore

```text
L_w^*L[f] = S.
```

In the transformed variable

```text
u = a^3 f,
```

the quadratic part becomes

```text
1/2 <L[f], L[f]>_w
  = 1/2 integral_0^1 (u')^2 dx.
```

The source coupling also simplifies:

```text
<S, f>_w
  = integral_0^1 S f a^4 dx
  = integral_0^1 S u a dx.
```

So the proposed variational target becomes

```text
E[u] = 1/2 integral_0^1 (u')^2 dx
       - integral_0^1 a S u dx.
```

The interior Euler-Lagrange equation in the transformed variable is therefore

```text
-u'' = aS.
```

Equivalently,

```text
L_w^*L[f] = S
```

becomes

```text
-u'' = aS
```

after multiplying by `a`. This is a strong algebraic simplification: the
candidate weighted variational problem is a disguised one-dimensional
Dirichlet/Poisson problem in `u`. It is still not a physical derivation of the
energy functional.

With the simple boundary test

```text
u'(0) = 0
u(1) = 0,
```

the Green kernel for

```text
-u'' = F
```

is

```text
G(x,y) = 1 - max(x,y).
```

Thus

```text
u(x) = integral_0^1 G(x,y)F(y)dy.
```

For the transformed source equation, `F = aS`. This makes the variational
target explicitly solvable once a domain is chosen.

However, regularity of

```text
f = u/a^3
```

at `x = 1` is much stronger than the minimal condition `u(1)=0`. It requires
high-order vanishing of `u` at the boundary. In the simple Green problem above,

```text
u'(1) = - integral_0^1 F(y)dy.
```

For positive source-family probes

```text
S_(p,q) = x^(2q)a^p,
F = aS_(p,q) = x^(2q)a^(p+1),
```

this integral is positive, so `u'(1)` is nonzero and negative. Therefore the
associated `f = u/a^3` is singular at `x=1` for these positive probes unless
additional cancellations, stronger boundary conditions, or a different domain
are supplied.

This turns regularity into a concrete admissibility direction:

```text
regular f requires moment/endpoint cancellations in F = aS.
```

For the Green problem with `u'(0)=0`, `u(1)=0`, the endpoint derivative
identities are:

```text
u'(1) = - integral_0^1 F dx
u''(1) = -F(1)
u'''(1) = -F'(1)
u''''(1) = -F''(1).
```

Since

```text
a = (1-x)(1+x),
```

boundedness of

```text
f = u/a^3
```

requires `u` to vanish to order at least 3 at `x=1`. With `u(1)=0` already
imposed, the first boundedness conditions are:

```text
u'(1)=0
u''(1)=0.
```

Using the endpoint identities, this becomes:

```text
integral_0^1 F dx = 0
F(1) = 0.
```

For the original source variable `F=aS`, this is:

```text
integral_0^1 aS dx = 0
(aS)(1) = 0.
```

The endpoint condition is automatic for regular `S`, because `a(1)=0`. The
global cancellation condition is not automatic:

```text
integral_0^1 aS dx = 0.
```

This is the first concrete admissibility condition produced by the transformed
energy problem. Positive source-family probes fail it; signed sources can
satisfy it.

A natural balanced source basis for this first condition is:

```text
B_q(x) = x^(2q) - c_q
```

where `c_q` is chosen so that

```text
integral_0^1 aB_q dx = 0.
```

The coefficient is:

```text
c_q = 3 / ((2q + 1)(2q + 3)).
```

Thus

```text
B_q(x) = x^(2q) - 3 / ((2q + 1)(2q + 3)).
```

The first member is:

```text
B_1(x) = x^2 - 1/5,
```

which is exactly the signed example that makes

```text
F = aB_1
```

pass the boundedness condition. The tested balanced-source projection
signatures retain full rank, so imposing the first admissibility condition does
not collapse the projection hierarchy; it replaces positive monomial probes by
signed balanced probes.

The boundedness condition is the base member of an endpoint-contact/admissibility ladder. For the
Green-domain problem,

```text
u'(1) = - integral_0^1 F dx
u^(n)(1) = -F^(n-2)(1),  n >= 2.
```

Since `F=aS`, endpoint contact of `f=u/a^3` translates into:

```text
R=0 bounded/non-contact level:
  integral_0^1 aS dx = 0

R=1 contact level:
  integral_0^1 aS dx = 0
  S(1) = 0

R=2 contact level:
  integral_0^1 aS dx = 0
  S(1) = 0
  S'(1) = 0

R contact level:
  integral_0^1 aS dx = 0
  S vanishes to order R at x=1.
```

An explicit source basis for the `R` endpoint-contact class is:

```text
B_(R,q)(x) = a^R [x^(2q) - c_(R,q)]
```

with:

```text
c_(R,q)
  =
  integral_0^1 x^(2q)a^(R+1) dx
  /
  integral_0^1 a^(R+1) dx.
```

This gives a concrete hierarchy of admissible signed source classes.

A finite row-space comparison gives an important guardrail. On even polynomial
source truncations, the projection rows

```text
P_k[S] = integral_0^1 psi_k S a^4 dx
```

are full-rank moment diagnostics. The low-order admissibility rows

```text
integral aS,
S(1),
S'(1),
...
```

have smaller rank until enough endpoint derivatives are included to span the
whole dual space. Therefore the `psi_k` hierarchy is not simply identical to
the low-order endpoint-contact ladder. It is adjacent to the admissibility problem,
and it resolves the balanced source spaces, but the row spaces do not match at
low regularity order.

This blocks the strongest possible bridge claim:

```text
psi_k hierarchy = endpoint-contact ladder
```

The better current statement is:

```text
psi_k hierarchy:
  full-rank moment diagnostics compatible with the admissible source classes

endpoint-contact ladder:
  boundary/domain constraints generated by the transformed energy problem
```

On the explicit balanced endpoint-contact source classes

```text
B_(R,q)(x) = a^R [x^(2q) - c_(R,q)],
```

the projection signatures

```text
P_R[k,q] = integral_0^1 psi_k B_(R,q) a^4 dx
```

have full tested rank for `R=0..5` and truncations through `N=7`. Thus the
projection hierarchy continues to resolve the admissible source classes after
the endpoint-contact constraints are imposed. The signature matrices are not
diagonal or triangular in the naive `q,k` ordering, so the relationship is a
full-rank diagnostic relation rather than simple mode-by-mode decoupling.

The balanced signature matrices factor as:

```text
P_R = M_R T_R,
```

where

```text
M_R[k,j] = integral_0^1 psi_k x^(2j) a^(R+4) dx
```

is the raw monomial projection matrix under the shifted weight, and `T_R` is
the balancing transform

```text
x^(2q) -> x^(2q) - c_(R,q).
```

So the admissible-source signatures are not a new projection mechanism. They
are the original projection mechanism composed with the endpoint-contact balancing
map.

Exact determinant checks show nonzero determinants for the original projection
truncations through `N=8` and balanced signature truncations for `R=0..5`
through `N=7`. This is still evidence rather than a closed determinant
formula, but it supports the full-rank diagnostic interpretation.

Finally, exact finite `psi_k`-adapted balanced bases can be constructed by
inverting the balanced signature matrix. In finite truncation, there are
balanced source combinations `D_l` satisfying:

```text
integral_0^1 psi_k D_l a^4 dx = delta_(k,l).
```

Thus the `psi_k` projections can serve as coordinates on finite balanced
admissible source spaces.

The natural variable for the projection algebra is also

```text
y = x^2.
```

In this variable:

```text
psi_k(y) = y^k - ((2k-1)/(2k+3))y^(k-1)
         = y^(k-1)(y-r_k),
```

and

```text
B_(R,q)(y) = (1-y)^R(y^q-c_(R,q)).
```

The pairing becomes a Jacobi-type polynomial pairing:

```text
1/2 integral_0^1
  psi_k(y)(y^q-c_(R,q))(1-y)^(R+4)y^(-1/2) dy.
```

The balancing coefficient has beta/Pochhammer form:

```text
c_(R,q)
  =
  B(q+1/2,R+2) / B(1/2,R+2)
  =
  (1/2)_q / (R+5/2)_q.
```

In `y` coordinates, `psi_1,...,psi_N` span an `N`-dimensional subspace of the
`N+1` dimensional polynomial space of degree `<=N`; there is exactly one
coefficient-space complement direction at each finite `N`. This supports the
current route toward a general determinant proof through polynomial filtration
and a codimension-one balancing constraint.

That missing coefficient direction has a closed form:

```text
m_j = 3 / ((2j+1)(2j+3)).
```

It is not incidental. The first admissibility functional on even polynomial
sources has coefficients:

```text
integral_0^1 a x^(2j) dx
  = 2 / ((2j+1)(2j+3))
  = (2/3)m_j.
```

Therefore, on finite even-polynomial coefficient spaces,

```text
span{psi_1,...,psi_N}
  =
ker[S -> integral_0^1 aS dx].
```

This is the strongest bridge found so far. The row functions themselves encode
the first admissibility condition at the level of polynomial coefficient
space.

For higher endpoint-contact classes, the kernel changes because the relevant moment
is weighted by `a^(R+1)`. The adapted basis is therefore:

```text
B_(R,q) = a^R[y^q - c_(R,q)],
```

rather than the unshifted `psi_k` family. Thus the exact `psi_k` kernel bridge
is strongest at `R=0`; higher endpoint contact is handled by the rebalanced
`B_(R,q)` source classes.

There is also a generalized row family adapted to the `R`-weighted
admissibility kernel:

```text
chi_(R,k)(y)
  =
  y^k - ((2k-1)/(2k+2R+3))y^(k-1).
```

The original rows are:

```text
psi_k = chi_(0,k).
```

This connects directly to the primitive-power family. The primitive-power
ratio is:

```text
(2k-1)/(2k+2m-1).
```

The endpoint-contact row ratio matches it when:

```text
m = R + 2.
```

Therefore the observed case

```text
m = 2
```

is the `R=0` member of the endpoint-contact ladder: the row family adapted to
boundedness of `f = u/a^3`. Higher endpoint-contact levels would naturally shift to
`m=3,4,...`.

Exact determinant pattern probes remain nonzero over the tested grids and
provide ratio/factor data, but no closed determinant product formula has been
proved yet.

If such a functional were independently supplied, the hierarchy could be read
as a finite-dimensional Galerkin or moment projection of the equation

```text
L_w^*L[f] = S.
```

The projection matrix would no longer be merely formal. It would be a
stiffness-like matrix associated with the energy problem.

To promote this from a speculative target to a result, one would need an
independent derivation of:

- the energy form `1/2 <L[f], L[f]>_w`;
- the weight `w = a^4`;
- the source pairing `<S, f>_w`;
- the allowed variation space;
- the boundary conditions that make the adjoint calculation clean;
- the reason this energy selects `L = L_2` rather than another `L_m`.

The zero-source case also needs care. Since

```text
L[f] = a^(-2) d/dx [a^3 f],
```

the condition

```text
L[f] = 0
```

implies

```text
d/dx [a^3 f] = 0.
```

Therefore

```text
a^3 f = constant
```

and

```text
f = constant / a^3.
```

This is singular at `x = 1` unless the constant is zero. Under regular
boundary conditions, the only source-free zero-energy profile in the `f`
variable is

```text
f = 0.
```

In the transformed variable `u`, the zero-energy profiles are constants, but
boundedness/contact in `f = u/a^3` forces that constant to vanish unless the domain or
baseline is changed.

Thus the variational picture does not by itself produce a nonzero trivial
profile. A nonzero baseline would require a different variable, a shifted
energy, a constraint, or a separately specified background state.
