# 1. Problem Statement

The current proof chain has reached a clean boundary.

It solved the original mystery:

```text
r_k = (2k - 1)/(2k + 3)
```

is the `R=0` boundary/admissibility moment coefficient.

It also reconstructed the GR branch conditionally: if exact metric response, diffeomorphism redundancy, Levi-Civita transport, stress coupling, second-order locality, boundary differentiability, and `D=4` are assumed, the standard GR branch is recovered in the expected scalar, tensor, radiative, boundary, constraint, and action sectors.

But this does not yet give a vacuum theory of gravity.

The achievement is sharper than a simple statement of incompleteness: the proof chain has localized the remaining field-equation problem to the strain/gradient sector of the vacuum configuration functional.

## What has been localized

The current ontology explains or motivates local response:

```text
configuration at a point -> directional interval response -> metric Hessian.
```

Field equations require between-point response:

```text
configuration mismatch between neighboring points -> strain energy -> transport law -> dynamics.
```

The remaining completion object is therefore:

```text
K_strain.
```

## Why this matters

Without the strain functional, the project can say:

```text
Given the GR-compatible metric branch, the known GR architecture follows.
```

With the strain functional, the project can attempt to say:

```text
The vacuum ontology generates field equations.
```

The next phase must specify, derive, or prove underdetermined the strain/gradient term that generates those equations.
