# 8. Field Equation Generation Target

The purpose of the vacuum strain functional is to generate field equations.

A successful next-stage theory should not merely say:

```text
If we choose EH/GHY, we get GR.
```

It should explain why the vacuum strain energy has the form it has.

## Target form

The target is a variational equation:

```text
δS_vac + δS_matter = 0.
```

With:

```text
S_vac = ∫ [ V_local + K_strain ].
```

The local part supplies the metric/interval response. The strain part supplies the dynamics.

## GR recovery target

The GR branch corresponds to:

```text
δ(S_EH + S_GHY + S_matter) = 0
```

which gives:

```text
G_ab + Λ g_ab = 8πG T_ab.
```

The vacuum theory must either derive this as the unique allowed branch or explain the allowed residuals.

## Extension target

A genuine extension must give field equations of the schematic form:

```text
G_ab + Λ g_ab + ε E_ab^residual = 8πG T_ab.
```

where `E_ab^residual` is not arbitrary; it must come from the vacuum strain functional and obey the relevant consistency conditions:

```text
conservation / Bianchi compatibility
boundary differentiability
causal propagation
no hidden source duplication
correct weak-field limit
explicit routing of extra modes
```

## Field-equation-generating axiom

The missing axiom is not simply “GR is recovered.”

The next axiom must determine `K_strain` strongly enough that variation produces either the GR equation or a controlled extension.
