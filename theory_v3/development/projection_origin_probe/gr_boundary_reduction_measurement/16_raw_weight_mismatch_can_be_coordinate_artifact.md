
# 16. Raw weight mismatch can be a coordinate artifact

Under the change

```text
y = z²,
```

the factor

```text
y^(-1/2) dy
```

pushes forward to

```text
2 dz.
```

Validated identity:

```text
(z²)^(-1/2) d(z²) = 2 dz.
```

So comparing raw displayed weights is not sufficient.  The invariant comparison
must track the full variable map, test space, and endpoint-contact class.
