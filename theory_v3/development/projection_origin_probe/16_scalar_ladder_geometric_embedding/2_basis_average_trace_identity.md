# 2. Basis Average Trace Identity

For an orthonormal directional probe basis, the average of the quadratic
responses

```text
Q(e_i) = e_i^T H e_i
```

recovers the trace channel:

```text
(1/3) sum_i Q(e_i) = Tr(H)/3.
```

The scalar projection is therefore naturally interpreted as an isotropic
average of directional interval probes.
