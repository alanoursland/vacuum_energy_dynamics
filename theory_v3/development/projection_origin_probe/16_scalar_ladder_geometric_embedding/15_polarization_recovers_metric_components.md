# 15. Polarization Recovers Metric Components

Directional probes reconstruct the full symmetric bilinear form. In two
dimensions,

```text
H_11 = Q(e_1),
H_22 = Q(e_2),
H_12 = (Q(e_1 + e_2) - Q(e_1) - Q(e_2))/2.
```

The script checks these identities exactly. This is the directional completion
that the scalar ladder lacks.
