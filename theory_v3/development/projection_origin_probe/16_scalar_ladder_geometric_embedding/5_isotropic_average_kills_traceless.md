# 5. Isotropic Average Kills Traceless Sector

Using the isotropic second-moment identity

```text
< v_i v_j > = delta_ij / 3,
```

the angular average of a quadratic response is

```text
< v^T S v > = Tr(S)/3.
```

For traceless `S`, this vanishes. The scalar/isotropic average therefore kills
traceless tensor structure by construction.
