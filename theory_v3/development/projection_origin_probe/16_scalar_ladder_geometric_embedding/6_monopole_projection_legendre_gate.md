# 6. Monopole Projection Legendre Gate

The scalar boundary ladder naturally matches the monopole projection. The
script checks

```text
int_{-1}^{1} P_l(x) dx = 0  for l >= 1,
int_{-1}^{1} P_0(x) dx = 2.
```

So the angular scalar average retains only the `l=0` mode. Higher multipoles
are not recovered by a scalar monopole ledger.
