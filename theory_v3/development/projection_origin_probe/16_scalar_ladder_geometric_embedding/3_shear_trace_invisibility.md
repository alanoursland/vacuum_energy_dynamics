# 3. Shear Trace Invisibility

The traceless shear response

```text
S = diag(s, -s, 0)
```

has

```text
Tr(S) = 0,
```

so the scalar trace channel sees nothing. But directional probes distinguish
it:

```text
Q(e_1) - Q(e_2) = 2s.
```

This proves that scalar projection cannot reconstruct shear even when the
shear is operationally visible to directional interval probes.
