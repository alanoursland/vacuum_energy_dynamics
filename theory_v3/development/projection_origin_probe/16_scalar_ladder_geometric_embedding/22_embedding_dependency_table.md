# 22. Embedding Dependency Table

This status file records the dependency structure.

```text
scalar admissibility ladder -> trace / monopole / scalar flux channel
quadratic directional probes -> symmetric bilinear metric data
polarization reconstruction -> off-diagonal and shear components
TT spin-2 branch -> radiative tensor polarizations
```

The script encodes the count distinction: one scalar trace coefficient is not
enough to cover either the full boundary tensor branch or the two-polarization
TT branch.
