# Vacuum Interval Directional Probe Origin: Selector-Level Plan

## Folder Role

This folder starts the first selector-level proof chain after
`vacuum_action_origin`.

The scalar projection ladder proved a strong admissibility structure, but it
does not contain enough rank to produce full tensor boundary data. The next
question is whether the vacuum interval/comparison ontology supplies the
missing directional data.

## Selector-Level Map

The remaining selector work naturally splits into four folders:

```text
vacuum_interval_directional_probe_origin
  prove how directional interval comparisons recover symmetric tensor data.

torsion_defect_exclusion
  prove torsion-free selection or isolate the source needed to activate torsion.

vacuum_dimension_selector
  derive, rather than only cross-check, why the physical branch is 3+1.

lambda_relaxation_mechanism
  derive zero-Lambda asymptotic flatness or a controlled nonzero-Lambda branch.
```

## First Target

This folder handles the first target:

```text
interval comparisons -> quadratic form -> symmetric bilinear tensor
```

If this closes, the action-origin chain has a route from the local interval
ontology to full induced boundary metric data. If it fails, the scalar
projection ladder remains auxiliary and the tensor boundary term must enter
from an external geometric assumption.

## Proof Batch 1

The first batch proves:

```text
1. A quadratic interval probe determines a symmetric bilinear form by polarization.
2. A finite axis-plus-pair probe set reconstructs all symmetric components in 3D.
3. A scalar trace probe is blind to the traceless sector.
4. Null-cone probes alone determine only conformal structure, not scale.
5. Boundary tangent interval probes recover induced metric data, not normal data.
6. Status: directional interval probes are necessary and sufficient for this gate.
```

## Proof Batch 2

The second batch should prove:

```text
7. Axis-plus-pair probes are sufficient in any finite dimension used here.
8. The recovered bilinear form has the correct local-frame covariance.
9. Null-cone data needs one non-null scale calibration to fix the metric scale.
10. Tangent plus normal probes recover the full bulk split at a boundary.
11. Boundary source/action coupling must use h_ab, not only tr(h), to see shear.
12. Status: the directional selector has a clean tensor-data gate.
```

## Proof Batch 3

The third batch should prove origin/metricity gates:

```text
13. The parallelogram law separates quadratic interval data from lower/odd contamination.
14. Multiple non-null calibrations must share one conformal scale.
15. Non-quadratic directional response fails bilinear metric reconstruction.
16. Isotropic direction averaging recovers only trace, not shear.
17. A full directional probe set recovers shear that isotropic averaging loses.
18. Status: metric interval origin requires quadraticity, common scale, and non-isotropic probes.
```

## Proof Batch 4

The fourth batch should prove second-variation origin gates:

```text
19. A local second variation / Hessian supplies the symmetric quadratic metric branch.
20. Small-displacement scaling separates quadratic response from cubic/quartic remainders.
21. A single local interval-scale field forces common conformal calibration.
22. Direction-dependent scale data is tensor/shear data, not a scalar conformal scale.
23. Nonquadratic finite-probe remainders can fake tensor components unless routed separately.
24. Status: Hessian origin is a viable conditional origin for metric-quality interval data.
```

## Closeout

This folder closes as a conditional tensor-boundary-data bridge:

```text
directional interval comparisons
  -> symmetric tensor data by polarization
  -> frame-covariant metric-like object
  -> induced boundary metric from tangent probes
  -> Hessian/second-variation origin if a local stationary vacuum functional exists.
```

The folder does not derive that local vacuum interval functional. That is a
future physical-origin task. Without such a functional, the result remains a
conditional selector bridge, not a completed field equation.
