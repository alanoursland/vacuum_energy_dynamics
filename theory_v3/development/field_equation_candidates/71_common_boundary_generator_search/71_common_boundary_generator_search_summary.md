## What Group 71 Established

### 1. Generator criteria are explicit

A legal common generator must do more than rename:

```text
B = D_jump + D_layer + D_tail
```

It must supply a boundary object before signs are chosen and produce:

```text
D_boundary = +B
L_boundary = -B
```

as opposite orientations of that object.

It must also avoid:

```text
repair current;
active O by label;
diagnostic transition insertion;
uncontrolled L_bulk / L_gauge leakage.
```

### 2. Orientation anti-match compatibility is explicit

Using:

```text
D_boundary = G
L_boundary = -sigma * G
```

the residual is:

```text
G * (1 - sigma)
```

Compatibility requires:

```text
sigma = 1
```

But Group 71 does not derive the orientation theorem. It only derives the sign a real opposite-orientation generator would need to force.

### 3. Component anti-match compatibility is explicit

Using:

```text
L_boundary =
  a_jump  * D_jump
+ a_layer * D_layer
+ a_tail  * D_tail
```

compatibility requires:

```text
a_jump = -1
a_layer = -1
a_tail = -1
```

The component result is not a geometry theorem. It also exposes `D_layer` as the dangerous component, because it must not be supplied by the old diagnostic transition response.

### 4. Bulk/gauge neutrality remains separate

Even ideal boundary anti-match leaves:

```text
L_bulk + L_gauge
```

Therefore the parent divergence identity still requires:

```text
L_bulk = 0
L_gauge = 0
```

or a lawful shared covariant-lift identity. Forced mutual cancellation is repair-like unless derived.

### 5. Generator classes were sieved

Retained only as theorem routes:

```text
pure common boundary generator;
boundary plus lift-boundary generator.
```

Rejected or blocked:

```text
renamed_B_only;
free_coefficient_generator;
repair_current_generator;
active_O_generator by label;
bulk/gauge contaminated generators without lift theorem;
layer-dependent generator without D_layer legitimacy.
```
