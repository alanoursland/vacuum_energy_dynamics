## Success Criteria

A source/divergence coefficient law succeeds only if it derives or structurally constrains:

```text
source no-double-counting sector-by-sector;
coefficient does not carry ordinary rho/M_enc load;
residuals do not carry ordinary source load;
boundary/support/layer terms do not carry ordinary source load;
divergence correction is explicit and non-reservoir;
divergence behavior does not hide source, boundary, current, mass, or support load;
coefficient behavior is not selected by recovery;
coefficient behavior is not selected by repair;
source/divergence constraints do not open parent equation prematurely.
```

---

## Failure Criteria

A route fails if it:

```text
hides ordinary source in coefficient;
hides source in zeta/kappa residual;
hides source in boundary/support layer;
hides source in correction tensor;
uses divergence correction as reservoir;
uses source leakage to select coefficient;
uses boundary leakage to select coefficient;
uses recovery success to select coefficient;
uses active O convenience to select coefficient;
uses parent-fit closure to select coefficient;
treats explicit divergence as divergence-safe theorem;
treats source discipline as B_s/F_zeta insertion;
adopts a Group 30 candidate postulate without explicit decision.
```
