# candidate_retention_contract — Result Note

## Result

The script defines the minimum contract for conditional audit retention.

Conditional retention requires future derivation of:

```text
u relation;
p_free;
trace/mass status;
ordinary-source neutrality;
covariant conservation/lift;
boundary behavior;
avoidance of active-O disguise.
```

## Main Findings

Conditional audit retention is allowed, but it is not free.

The candidate may remain as obstruction-tagged audit material only if the unresolved burdens remain explicit. Otherwise retention becomes candidate hoarding.

The script rejects:

```text
candidate hoarding;
principle by label;
O disguise.
```

This is important. Saying “maybe variational” or “maybe geometric” is not enough. A future group would need to actually derive the missing structure.

## Boundary

The retention contract does not solve the obstruction. It names the burdens required for the candidate to remain live.

## Steering Consequence

The next script should separate legitimate future escape routes from shortcut escapes.
