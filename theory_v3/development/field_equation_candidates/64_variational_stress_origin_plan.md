### 3. candidate_quadratic_scalar_origin.py

Test whether:

```text
eta=(1-y^2)^2*(y-c)
```

can satisfy a constant-coefficient quadratic Euler-Lagrange equation:

```text
f'' - k f = 0
```

If `f''/f` is not constant, this simple origin fails.

### 4. candidate_linear_closure_origin.py

Test a linear closure:

```text
u=a p_r+b p_t
P=p_r+2p_t
T=-u+P
A=u+P
```

Show that trace-free and active-mass-neutral requirements still demand incompatible closures unless `P=0`.

### 5. candidate_amplitude_origin_scale.py

Audit `p_free`.

Reject:

```text

diagnostic-repair p_free;
source-coupled p_free;
zero p_free.
```

A normalization can define units, not derive physics.

### 6. candidate_boundary_variation_sieve.py

Test natural-boundary behavior for a local quadratic functional. Endpoint value/slope silence helps, but natural-boundary satisfaction is not a full variational origin unless the interior Euler-Lagrange equation also holds.

### 7. candidate_variational_route_classifier.py
