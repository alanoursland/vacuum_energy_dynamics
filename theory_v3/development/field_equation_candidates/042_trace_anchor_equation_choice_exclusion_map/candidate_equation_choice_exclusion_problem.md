# Candidate Equation Choice Exclusion Problem

## Result

The script opens Group 42 as an equation-choice exclusion map.

It allows candidate equation families to be classified, eliminated, demoted, or marked axiom-required before any new axiom or branch choice is made.

## Main Findings

- Equation elimination is allowed, but equation selection is not.
- The reduced A-sector areal-flux law remains the protected reduced mass-response anchor and may constrain candidate equations.
- Surviving equation families may remain only conditional candidates or axiom-required future candidates.
- Axiom need may be named, but no axiom is adopted.
- `B_s/F_zeta` insertion, active `O`, residual control, Package B adoption, and parent closure remain closed.

## Boundary

This opener does not choose, adopt, derive, declare, insert, or open parent closure. It rejects treating surviving candidates as selected equations, treating axiom need as axiom adoption, using recovery as a selector, using undefined objects as repairs, or treating exclusion mapping as parent readiness.
