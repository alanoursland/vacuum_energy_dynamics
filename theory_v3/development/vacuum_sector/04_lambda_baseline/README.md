## Variational-Minimum Probe

The variational-minimum probe tests whether bare stationarity over `Lambda`
selects a nonzero baseline. It does not. With the current inputs, the route
leaves `Lambda` free, selects `Lambda = 0`, or produces a nonzero value only by
importing a bias, target, boundary class, or new scale. This probe tests
selector functionals `F(Lambda)`, not variation of the full EH action with
respect to `Lambda`.

## Boundary/Admissibility Probe

The boundary/admissibility probe tests whether explicit boundary data can
supply the missing nonzero scale. It records that boundary data can convert a
supplied length, radius, volume, or asymptotic class into a `Lambda` relation,
but does not derive that supplied scale.

## Topology/Global Constraint Probe

The topology/global constraint probe records that topology can restrict sectors
or relate `Lambda` to supplied area, volume, length, or measure data, but
topology alone is dimensionless and cannot set a dimensionful `Lambda`.

## Measure Identity Probe

The measure identity probe records that a measure route can become a Lambda
selector only if it derives a covariantly conserved constant floor before
observation is used. Dimensional fits import a scale, and variable or clustered
densities belong to dark-sector or defect ledgers.

## Relaxation/Fixed-Point Probe

The relaxation/fixed-point probe records that scale-free relaxation selects
`Lambda = 0`. Nonzero fixed points import a target, domain length, kernel
scale, or coefficient ratio unless those quantities are derived.

## Frustration-Floor Microphysics Probe

The frustration-floor microphysics probe records that potential shape can
supply minima and excitations, but the absolute constant offset remains free
unless microphysics derives it. Excitations around the floor route downstream
to dark-sector or defect ledgers, not the constant `Lambda` baseline.
