# Appendix C. Sample Lab-Report Structure

```text
Title:

Trial group:

Charter reference:

Question:

Inputs and dependencies:

Script artifacts:

Derivation summary:

Checks performed:

Failure controls:

Verdict:

What was established:

What was not established:

Downstream handoff:

Supersession notes:
```

The report should distinguish executable checks from interpretation. A script
may establish an algebraic identity while leaving the physical scope open.
