# 7. Selection Theorem

The selection theorem combines the previous two lemmas.

1. Any nonzero `a` in `R + a R^2` gives a scalaron with range
   `ell = sqrt(6a)`.
2. Any nontrivial sourced static body has mandatory exterior scalaron hair for
   `a != 0`.
3. Any exterior scalaron hair carries temporal-radial anisotropy and violates
   `AB = 1`.
4. Static frame indifference requires `AB = 1` in static source-free exteriors.

Therefore:

```text
a = 0 .
```

Within the healthy local `R + a R^2` family, static frame indifference selects
the Einstein-Hilbert action with a cosmological term:

```text
S = c^4/(16 pi G) int sqrt(-g) (R - 2 Lambda) d^4x + S_matter
```

plus inert Gauss-Bonnet freedom in four dimensions.

The selection is principle-based, not fit-based. Short-range experiments are
not used to choose `a = 0`. Instead, they become tests of the principle: a
gravitational-strength scalar Yukawa detection would force an appeal or
revision of static frame indifference.
