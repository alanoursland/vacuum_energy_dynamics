# Appendix D. Yukawa Parameter Translation

The weak-field potentials of `R + a R^2` gravity around a point source are

```text
phi(r) = -(GM/r) [1 + (1/3) exp(-r/ell)],
psi(r) = -(GM/r) [1 - (1/3) exp(-r/ell)].
```

The scalaron range is

```text
ell = sqrt(6a).
```

The Newtonian potential therefore receives a Yukawa correction with

```text
alpha = 1/3,
lambda = ell.
```

In the current data notes, the Lee 2020 vector-extracted curve gives the
`alpha = 1/3` crossing at

```text
lambda = 54.05 um.
```

This appendix should be updated if official author-provided limit tables
replace the vector-extracted curve.
