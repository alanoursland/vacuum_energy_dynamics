# 1. Introduction

Higher-curvature gravity is usually constrained by dynamics and observation.
In the local four-derivative sector, ghost freedom removes terms that modify
the transverse-traceless spin-2 propagator. In four dimensions,
Gauss-Bonnet is inert. The familiar healthy survivor is the `R^2` term,
equivalently a massive scalar degree of freedom in the scalar-tensor
description.

This note asks whether a static selection principle removes that survivor.
The principle is static frame indifference: a strictly static vacuum exterior
should not carry an intrinsic preferred temporal-radial frame. In static
spherical areal gauge, its metric shadow is

```text
A(r) B(r) = 1 .
```

Schwarzschild and Schwarzschild-de Sitter satisfy this condition exactly. A
generic `f(R)` star exterior does not.

The argument has three parts. First, any nontrivial static scalar curvature
profile in `R + a R^2` carries a temporal-radial anisotropy. Second, for a
sourced body, scalaron hair is mandatory for any nonzero `a`. Third, mandatory
hair plus static frame indifference is a contradiction, so `a = 0`.

The paper is written mostly in standard modified-gravity language. The
motivation for the principle comes from Vacuum Energy Dynamics, where
`AB = 1` is the metric shadow of a substance statement: static vacuum carries
no energy current and no preferred frame in the t-r plane. The theorem itself
can be read independently as a selection rule for the healthy local `f(R)`
family.

The novelty check is not complete. The final version must position the result
against known `f(R)` exterior solutions, no-hair theorems, and screening
literature. The provisional claim is not that `f(R)` hair is unknown, but that
mandatory hair plus static frame indifference gives a compact selection theorem
for `a = 0`.
