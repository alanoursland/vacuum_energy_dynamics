# 11. Conclusion

The `R^2` scalaron is normally treated as a healthy extension constrained by
experiment. Under static frame indifference, it is eliminated before fitting.

The argument is compact. Nonzero `a` gives a scalaron. Sourced static bodies
must leak scalaron hair. Scalaron hair carries temporal-radial anisotropy and
violates `g_tt g_rr = -1` in areal gauge. Static frame indifference forbids
that violation. Therefore `a = 0`.

The result does not rule out all modified gravity. It selects the
Einstein-Hilbert action from the healthy local `R + a R^2` family under a
specific static-vacuum principle. If the principle is kept, the short-range
Yukawa channel is a null test. If the principle is dropped, experiments bound
the scalaron range instead.
