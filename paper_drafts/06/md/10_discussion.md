# 10. Discussion

The central question is whether static frame indifference is too strong. Many
modified-gravity theories intentionally add scalar structure to static
exteriors. This paper does not rule them out by ordinary consistency alone. It
rules them out only if the selection principle is imposed.

Several routes evade the theorem:

- abandon static frame indifference;
- restrict the principle to the two-derivative sector only;
- permit `AB != 1` within a finite scalaron layer near matter;
- use nonlocal modifications outside the local `R + a R^2` family;
- use screening models whose assumptions evade the simple matching witness;
- consider nonstatic, rotating, nonspherical, or cosmological settings.

These are not loopholes in the algebra. They are choices about the principle's
scope. If one accepts exact static frame indifference as a physical selection
rule for static vacuum, then scalaron hair is not merely constrained. It is
forbidden.

For Vacuum Energy Dynamics, this is the role of the theorem: P7-prime forbids
preferred temporal-radial frames in static vacuum, and the scalaron necessarily
creates one. For readers outside VED, the theorem can be read conditionally:
`AB = 1` as an exact static-vacuum principle selects `f(R) = R` from the
healthy local `R + a R^2` family.
