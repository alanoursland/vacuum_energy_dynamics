# Appendix C. No Positive Root of `x cosh x = sinh x`

Define

```text
h(x) = x cosh x - sinh x .
```

Then

```text
h(0) = 0 .
```

Differentiating gives

```text
h'(x) = cosh x + x sinh x - cosh x = x sinh x .
```

For every `x > 0`,

```text
x sinh x > 0 .
```

Therefore `h` is strictly increasing on positive `x`. Since `h(0) = 0`,

```text
h(x) > 0    for all x > 0 .
```

Thus `x cosh x = sinh x` has no positive solution.
