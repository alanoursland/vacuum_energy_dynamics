# Appendix C. Radiative Bootstrap

The radiative bootstrap fixes the transverse-traceless normalization that the
linear theory leaves free.

Linear massless spin-2 theory supplies the gauge-invariant operator but leaves
the kinetic coefficient `K_T` undetermined. P9 then requires the field's own
energy to source the response at the same universal coupling as matter.

The resulting values are:

```text
N = c^4 / (8 pi G)
K_T = c^4 / (16 pi G)
<u> = c^2/(32 pi G) <dot h_ij dot h_ij>
P = (G / 5 c^5) <dddot Q_ij dddot Q_ij>
```

steps from the verification scripts. For the current draft, the main text
states the result and points to Paper 2 for the proof core.
