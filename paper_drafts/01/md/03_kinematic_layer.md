# 3. Kinematic Layer: Why the Vacuum Configuration Is a Metric

P2 identifies the vacuum configuration with spacetime. Operationally, this
means that clocks and rods read the local vacuum mappings. A perturbation of
the vacuum configuration therefore appears to matter as a perturbation of
intervals.

At the local zero-source state, the imported special-relativistic structure is
Minkowski. A small departure from that state is represented by a symmetric
tensor perturbation `h_ab` of the interval map:

```text
g_ab = eta_ab + h_ab .
```

The symmetry of `h_ab` is not an extra gravitational assumption. It follows
from the fact that the interval is a symmetric bilinear object. The universal
matter response is also not an independent equivalence-principle axiom in this
language: matter is built from processes that read the same vacuum mappings,
so all matter actions couple to the same `g_ab`.

Relabeling vacuum elements is a gauge freedom. If the identity of a vacuum
element is changed by an infinitesimal displacement `xi_a`, physics cannot
depend on that labeling choice. The perturbation therefore has the familiar
linear gauge transformation

```text
h_ab -> h_ab + partial_a xi_b + partial_b xi_a .
```

This is the kinematic entry point for spin-2 field theory. The theory has not
yet assumed the Einstein tensor or Einstein-Hilbert action. It has established
a symmetric tensor response variable, universal matter coupling, and relabeling
gauge freedom from the vacuum-spacetime identity.

This section is deliberately modest. It does not claim to have derived the
full differential-geometric apparatus of GR from ontology alone. It establishes
the kinematic conditions that any local response law for the vacuum metric
must satisfy.
